## Features

1. Provide a script to update the documentation vector database during documentation deployment.
1. Support quick attribute input in the expression editor.
1. Add 5-day and 1-day license expiration reminders.
1. Allow direct editing of sample connection settings when TDengine is unreachable.
1. Support bulk creation of element templates by uploading a CSV file.
1. Keep expression styling consistent with other pages.
1. Optimize AI service memory consumption.
1. Improve Enter key behavior in AI Q&A and add Shift+Enter for line breaks.
1. Improve SQL processing in AI Q&A through multiple AI interaction rounds.
1. Support returning chart data in AI Q&A responses.
1. Initialize panel limit settings based on selected attributes.
1. Allow manual column width adjustment on list pages.
1. Improve event breadcrumb menu display.
1. Add a context menu to the element template tree view.
1. Support specifying a referenced attribute template name in attribute template data reference settings.
1. Optimize AI service configuration.
1. Align attribute template names with virtual supertable column names.
1. Add a Generating status to sample tasks.
1. Support returning progress information from the sample backend.

## Improvements

1. Use individual `last` queries to retrieve attribute values, fixing inaccurate attribute value update timestamps.

## Bug Fixes

1. Fixed sample data loading failure in IDMP cloud service caused by database permission synchronization.
1. Fixed a backend exception thrown by the AI service when an analytics update produces output attributes.
1. Fixed event bookmarking issues.
1. Fixed incorrect help link URLs in AI Q&A responses.
1. Fixed missing alias when sorting panels.
1. Fixed inability to set an OpenAI endpoint URL in AI connection settings.
1. Fixed failure to import composite metrics.
1. Fixed "AI service busy, please try again later" errors.
1. Fixed incorrect column names in state window analytics generated by AI.
1. Fixed mismatch between attribute template names and actual virtual supertable column names.
1. Fixed inability to select attributes in panel filter settings.
1. Fixed requirement to always select the first metric in panel window settings.
1. Fixed array index out-of-bounds error when importing data with multiple trees.
1. Fixed the `idmp generator` command-line tool failing to execute.
