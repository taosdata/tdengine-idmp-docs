## Features

1. Support silent login for standalone single sign-on.
1. Provide a client SDK for third-party integrations.
1. Panels and dashboards can be shared with third parties.
1. Support state history panels.
1. Add a filter function to the element template page.
1. Complete a major refactor of the AI service.
1. Add a normalization button to event trend panels.
1. Add the ability to delete test events.
1. Add energy and IoT symbol libraries to SCADA panels.
1. Element event lists can be saved as panels.
1. Automatically create TDengine tag data references for general attributes and element categories.

## Improvements

1. Add the full element path to child element lists, attribute lists, and similar pages.
1. Complete optimization of the SCADA panel editing interface.
1. Support both username/password and single sign-on for the same email address.
1. Attribute template list page supports pagination.
1. Panel and panel template names must be unique within the same element or element template.
1. Improve event notification rules.
1. Optimize sorting on panel and panel template list pages.
1. Allow special characters in element and element template names.
1. Allow deleting attributes from within an attribute's trend chart.
1. Align element category panel and search UI for consistency.
1. Improve the asset list panel functionality.
