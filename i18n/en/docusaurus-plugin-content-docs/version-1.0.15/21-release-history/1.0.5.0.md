## Features

1. New attribute templates use the keyword entered when creating elements.
1. Support filtering child element lists by element template name.
1. Allow users to specify the display order of child elements and attributes.
1. Panels support state timeline charts.
1. All list pages support exporting the current page.
1. Event trend panels support saving and editing.
1. Analytics support referencing formula and string constructs.
1. User login supports an optional CAPTCHA verification step.
1. Units of measure support search and browsing of search results.

## Improvements

1. Panel metric axes display the current metric's physical unit.
1. Import/export allows opting out of uploading the taosgen file.
1. Panel auto-refresh should not reset the already-selected time zoom range.
1. Batch retrieval of the latest values for attribute lists.
1. Prediction values are shown only in the metric's own swimlane.
1. Adjust the notification rule resend interval to 8 hours.
1. Change the default English email server to AWS.
1. Data import supports specifying element templates for intermediate nodes.
1. Attribute list adds an icon indicating the data reference type and moves the type column to the fifth position.

## Bug Fixes

1. Fixed AI service connection timeout to the vector database.
1. Fixed element template naming patterns containing special characters causing element files to not be found.
1. Fixed a name-object mismatch when editing UOM categories.
1. Fixed repeated simple import runs invalidating previously created analytics.
1. Fixed an error when clicking Evaluate in the event window trigger during analytics editing.
1. Fixed data import failing when there are too many child tables.
1. Fixed panels generated from AI-recommended questions on intermediate-node elements not displaying data correctly.
1. Fixed anomaly detection analytics not indicating required output attributes in the UI.
1. Fixed missing validation of element template name requirements.
1. Fixed copying a dashboard template to an element template with no panel templates failing.
1. Fixed searching for an element by name returning multiple elements with the same name.
1. Fixed incorrect start event notification links in event window analytics.
1. Fixed dashboard editing causing line breaks to appear on the dashboard page.
1. Fixed missing frontend validation for empty start and end conditions in event window analytics.
1. Fixed map-type panels not displaying correctly.
1. Fixed inability to import unsigned data types.
1. Fixed failure when importing element templates.
1. Fixed element template import reporting "Naming pattern is invalid" errors.
1. Fixed successful data imports not showing the file name in the import task result list.
