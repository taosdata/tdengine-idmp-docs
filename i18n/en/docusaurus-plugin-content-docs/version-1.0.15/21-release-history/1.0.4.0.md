## Features

1. Panel expressions support directly using formula attributes.
1. Proactive AI insights support recommending and generating anomaly detection analytics.
1. Support copying event trend charts.
1. Advanced SQL in panels supports substitution parameters.

## Improvements

1. Python version requirement updated to 3.12.
1. Improve the email activation code flow.
1. Improve the connection information form on the sample selection page.
1. Add display of connection status and detailed error messages.
1. Search supports case-insensitive matching.
1. Trend chart statistics support hide, table, and row display modes.
1. Event search results page display is consistent with the main event list page.
1. Optimize the JSON definition of element attributes.
1. Panel scatter plots support comparing X-axis and Y-axis metrics.
1. Generate recommended questions based on the user's activation language.
1. Improve complex metric generation efficiency.
1. Make analytics output attributes optional.
1. CSV files for download samples and documentation only support UTF-8 encoding.
1. Panels support setting Y-axis value range during editing.
1. Panels support `tbname` as a dimension.

## Bug Fixes

1. Fixed elements being able to create a reference to themselves within the same tree.
1. Fixed AI Q&A responses containing Chinese text in English environments.
1. Fixed asset model import not being able to use already-created templates.
1. Fixed AI Q&A requiring a 2-minute wait even after an error.
1. Fixed AI Q&A failing to answer questions about newly imported element templates.
1. Fixed AI Q&A failing to answer questions about "the last 7 days."
1. Fixed removing the interval time from a sliding window analytics having no effect.
1. Fixed elements not inheriting notification rules set on element templates by default.
1. Fixed data import failing when child table tags are empty.
1. Fixed event analytics not sending notifications.
1. Fixed intermediate-node elements returning incorrect data in panels after being renamed.
1. Fixed incorrect data limits when grouping and sorting are enabled in a panel.
1. Fixed AI-generated analytics occasionally failing to save.
1. Fixed attribute template type changes not propagating to the corresponding element attributes.
1. Fixed inaccurate trigger conditions in AI-generated analytics.
1. Fixed analytics output attributes failing to update data references in some cases.
1. Fixed inability to send emails to third-party mail servers.
1. Fixed AI Q&A reporting a "func does not exist" error.
1. Fixed inability to view events after the associated analytics is deleted.
1. Fixed failure to delete analytics when deleting an element.
1. Fixed data import not correctly importing panels.
1. Fixed syntax errors in AI Q&A.
1. Fixed AI-recommended panel questions failing to generate panels.
1. Fixed local DeepSeek AI connection configuration not taking effect.
1. Fixed failure when moving an element to a child of its current parent.
1. Fixed AI-generated panels reporting "Aggregation functions do not support nested" errors.
