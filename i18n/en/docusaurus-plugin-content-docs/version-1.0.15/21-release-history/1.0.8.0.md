## Features

1. Element and attribute list pages can be saved as panels.
1. Analytics templates support child element aggregation.
1. Improve intent recognition in AI Q&A.
1. Support attribute search.
1. Analytics support recalculating historical data.
1. Support SCADA panels. (Preview)
1. Fixed AI Q&A failing to find elements.

## Improvements

1. Improve accuracy of complex metrics generated by AI.
1. Show element count on the element template list page.
1. Improve element matching in AI Q&A.
1. Apply summarization-based vector optimization to documentation-type AI Q&A answers.
1. Remove global locks and implement per-analytics resource locks.
1. Change element attribute template updates to asynchronous operations.
1. Do not display units for attributes with null values.

## Bug Fixes

1. Fixed some attributes having no queryable values after asset import.
1. Fixed invited users being unable to expand the root element.
1. Fixed element template import appending "+1" to names.
1. Fixed inability to upload images in rich text panels.
1. Fixed failure to upload background images in rich text panels via the widget uploader.
1. Fixed CSV-imported element templates causing issues with virtual tables.
1. Fixed a panel not displaying correctly when the same attribute is used as both a metric and a dimension.
1. Fixed asset model import failing to configure child table filter Not Null rules.
1. Fixed inability to delete element templates.
1. Fixed dashboards not displaying decimal values.
1. Fixed attribute data length being inconsistent with TSDB.
1. Fixed panels that include child element attributes being convertible to templates incorrectly.
1. Fixed child element template aggregation panels not displaying data in dashboards.
1. Fixed stat panels not displaying decimal values.
1. Fixed event trend charts not displaying time ranges correctly.
1. Fixed rich text panel templates always showing the "Title" string.
1. Fixed events not displaying microsecond timestamps.
1. Fixed unclear error messages when a panel template ID is not found.
1. Fixed user profile completion auto-filling the password field.
1. Fixed microsecond handling issues in event detail and duration display.
1. Fixed incorrect links added to event notification email details.
