## Features

1. Attribute value types support the Object type.
1. Panels and dashboards support scheduled delivery.
1. Panels support unit inference.
1. Data prediction supports multivariate correlation analysis.
1. Panel metrics and tags each support adjusting display order independently.
1. Advanced panels support uploading SVG symbols.
1. CSV import supports attribute and element categories.
1. Navigation bar changed from a dropdown menu to a flat layout.
1. Event lists can be saved as panels.
1. Element search supports limits and sorting.
1. Element search supports specifying a root element.
1. Support reverse proxy access.
1. Provide an Excel Add-in component.

## Improvements

1. Panels support automatic downsampling.
1. Index read-dirty-data capability enhanced for transactional processing.
1. Simple import supports select-all and deselect-all for asset models.
1. Improve reverse proxy support.
1. Excel Add-in fixes several issues.
1. Element search supports selecting the sort attribute.
1. Add upgrade documentation for local installation environments.
