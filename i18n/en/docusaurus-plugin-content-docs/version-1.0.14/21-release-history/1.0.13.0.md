## Features

1. Panels support using elements as dimensions.
1. Add support for panel metric statistics.
1. Element templates support inheriting from multiple base templates.
1. Add element documents as reference material for panel and analytics recommended questions.

## Improvements

1. OPC import tasks add a listener task to continuously monitor for new table creation.
1. Change the panel Y-axis unit of measure display to abbreviated form.
1. Other panel types also display units of measure by default.
1. Add column display configuration to the element template list page.
1. Move notification rules from the element path bar to the action buttons on the event list.
1. Clicking an attribute name in attribute search results navigates to the attribute detail page.
1. Improve display of multiple dashboards in child element aggregation panels.
1. Excel Add-in asset filter can list element paths.
1. OPC-UA import supports importing from multiple databases simultaneously.
1. Improve multiple user interactions for enum set deletion.
1. Improve the attribute data reference storage structure.
1. Support AI interpretation while editing panels.
1. Element template attribute template lists can display attribute templates from base templates.
1. Optimize attribute storage structure and remove parts duplicated from attribute templates.
1. Add batch delete APIs for element and event template lists in the backend.
1. Analytics can be directly managed from the admin backend.

## Bug Fixes

1. Fixed OPC-UA import cancel button not working.
1. Fixed AI service memory growth issues.
1. Fixed event generation issues in anomaly detection windows.
1. Fixed copying analytics producing duplicate stream computation names.
1. Fixed no response when switching login between users with different permissions.
1. Fixed exceptions during OPC-UA import.
1. Fixed AI service becoming unresponsive and then recovering after a restart.
1. Fixed AI Q&A failing to output answers.
1. Fixed element naming templates not supporting attribute value substitution parameters.
1. Fixed multiple display issues in SCADA panels.
1. Fixed CSV import not generating new elements for newly added child tables.
1. Fixed incorrect Explorer link addresses in connection information.
1. Fixed shared panels switching the currently logged-in internal user when opened in the same browser.
1. Fixed inability to update virtual tables when clearing the source table and source column of data reference attributes.
1. Fixed the editing area in panel edit being too small.
1. Fixed cloud service shared dashboards failing to load.
1. Fixed a tooltip flashing in the top-left corner when toggling the show/hide config panel button in panel editing.
1. Fixed inability to register a new account when an existing user already exists.
1. Fixed importing elements from CSV making the AI service unavailable.
1. Fixed the element analytics list returning analytics template names.
1. Fixed SCADA panel background images not displaying after upload.
