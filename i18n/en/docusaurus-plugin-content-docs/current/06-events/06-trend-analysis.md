When an event occurs, understanding the data behavior around that event is essential for investigation. TDengine IDMP provides a direct path from any event to a trend chart pre-configured with the event's time range.

## 6.6.1 Opening a Trend Chart from an Event

You can open a trend chart for any event in two ways:

**From the event list:**

Hover over an event row to reveal the **⋮** menu, then click **Trend Chart Analysis**. IDMP opens the associated element's trend chart, automatically zoomed to the event's start time and end time.

**From the event detail page:**

On the General tab, click the **Trend Chart Analysis** icon in the toolbar to open the same view.

## 6.6.2 Working with the Event Trend Chart

The trend chart opened from an event shows the element's time-series attributes over the event's time window, with the event's start and end times marked on the chart.

Multiple events can be added to the same trend chart analysis, allowing you to overlay and compare different events side by side — for example, comparing the sensor behavior during two separate fault occurrences.

From the chart view you can add additional attributes from the element or from other elements in the asset tree, and zoom or pan to adjust the time window.

:::note
Trend Chart Analysis is a powerful feature with many capabilities. A dedicated section in the **Advanced Analytics** chapter covers it in full detail.
:::
