# Examples

Complete, runnable code samples for real-world scenarios. Copy any example and adjust it to your environment.

**Before running any example, set:**

```bash
export IDMP_HOST=http://your-idmp-server:6042
export IDMP_USERNAME=your_username
export IDMP_PASSWORD=your_password
# or if you already have a token:
export BEARER_TOKEN=your_token
```
