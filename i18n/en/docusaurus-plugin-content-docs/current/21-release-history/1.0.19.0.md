**Release Date: 2026-05-31**

## Features

1. For attributes of element reference type, support for configuring reference relationships is now available, making it easier to establish associations between elements.
1. The panel list and panel template list now support export and import, enabling easy configuration migration and reuse.
1. The panel now supports heatmaps for visualizing data density and distribution patterns.
1. Redis integration completed.
1. User authentication now supports LDAP/AD integration for centralized enterprise identity and permission management.
1. The dashboard editing page has been refactored for smoother operations and a significantly improved editing experience.
1. Version control now supports the "Review Required" mode, enabling mandatory change review before deployment for better collaboration control.
1. A single panel can now include trend charts, bar charts, state history charts, and state timeline charts simultaneously, meeting multi-dimensional visualization needs.
1. Add AI document processing progress display in document list and element document list.
1. Admin backend now supports user groups for centralized member permission and role management.
1. Panel now supports candlestick charts.
1. Internal database supports MySQL.

## Improvements

1. The background color of standard panels now supports customization, including transparent and any other color.
1. Refactor panel configuration items, add candlestick, heatmap and histogram support.
1. Improve attribute search performance.
1. Table panel supports auto-paging.
1. State/history charts now support popup and series config.
1. When sharing panels and dashboards, you can hide the time picker for a cleaner and clearer shared view.
1. Date format automatically adapts to your browser's locale.
1. Dashboard and event main menu page now support filtering by template for quick content location.
1. CORS and refresh token security issues.

## Bug Fixes

1. Fixed the issue where a vertical scrollbar incorrectly appears on the right side of the chart area in trend and bar charts.
1. Add timezone restriction to analysis panel data query time conditions.
1. Add create element group to root element group's more menu.
1. When editing a panel template under an element, exiting without any changes incorrectly creates a new panel.
1. AI connection URL validation incorrectly reports 404 for local service addresses.
1. Activation email failure incorrectly shows success message.
1. Support Azure OpenAI proxy by adding api-version and removing incompatible parameters.
