**Release Date: 2025-11-15**

## Features

1. AI connection defaults to the ChatGPT endpoint in the English version.
1. The proportion of complex metrics in proactive AI insights is configurable.
1. Event trend charts support data gap filling.
1. Import/export supports backend API-driven export of specified elements and element templates.
1. Proactive AI insights include questions for generating multi-metric anomaly detection analytics.
1. Formula attributes support aggregate calculations.

## Improvements

1. AI Q&A documentation queries integrate keyword search.
1. TSDB queries support retry.
1. Enum-type attributes are temporarily not allowed to reference TDengine data.

## Bug Fixes

1. Fixed newly created AI connections not appearing in the list.
1. Fixed sample generation reporting a "network packet exceeds maximum size" error.
1. Fixed incorrect links in AI Q&A answers.
1. Fixed AI Q&A questions not being copyable.
1. Fixed AI Q&A answers sometimes appearing truncated.
1. Fixed panel limit value settings not taking effect immediately.
1. Fixed panel legends not displaying inside dashboards.
1. Fixed the AI-generated analytics for event generation not having "generate event" selected on the edit page.
1. Fixed inability to add analytics templates from elements to event trend charts.
1. Fixed inability to add new AI connections.
1. Fixed Cloud IDMP instances failing to connect to Cloud TSDB.
1. Fixed invited users completing their profile in English inadvertently changing the system default language.
1. Fixed timed window analytics potentially producing duplicate calculations.
1. Fixed inability to import dashboards on elements.
1. Fixed time range control issues in event trend charts.
1. Fixed incorrect "acknowledgement required" settings in notification emails.
1. Fixed attribute list displaying timestamps with microsecond precision by default.
1. Fixed concurrent panel queries returning 403 unauthorized errors.
1. Fixed rounding functions such as `TRUNCATE` not displaying results correctly in dashboard panels.
1. Fixed the element selected in dashboard templates not being passed through to panel templates.
1. Fixed AI Q&A failing to include parent element attributes.
