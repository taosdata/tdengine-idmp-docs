**Release Date: 2025-09-06**

## Features

1. Complete basic import and export functionality.
1. Panels support map chart type.
1. Support element drag-and-drop in the element browser.
1. Panels support scatter plots.
1. Panels support advanced SQL.
1. Support creating anomaly detection analytics.

## Improvements

1. The element selector in panel templates can adapt its width to the browser viewport.
1. Improve the analytics deletion prompt to warn about deleting old data first.
1. Improve storage and usage of attribute template references in element attributes.
1. Use DeepSeek V3 for recommended questions.

## Bug Fixes

1. Fixed data template import using the same path with different element templates generating entries under different paths.
1. Fixed failure to update the connection when TSDB is unreachable on the samples page.
1. Fixed inability to modify custom roles.
1. Fixed the same event template appearing twice in notification rules.
1. Fixed data import failing immediately when the configured element template does not exist.
1. Fixed covariates being selectable in data prediction only for the Moirai model.
1. Fixed AI-generated analytics missing window aggregation settings.
1. Fixed inability to delete only referenced elements.
1. Fixed AI Q&A generating unsupported SQL causing direct errors.
1. Fixed event filter conditions having no effect.
1. Fixed mismatch between AI-generated analytics output attribute names and the created result table.
1. Fixed panel generation failures causing the AI service to become busy.
1. Fixed AI Q&A giving poor answers to statistical questions.
1. Fixed bar charts not displaying when there is only one column of data.
1. Fixed inability to continue asking questions after generating a panel from the panel list AI Q&A.
1. Fixed analytics output attributes failing to update data references in some cases.
1. Fixed inability to import unsigned data types.
1. Fixed the dimension table not displaying when adding dimensions to panels on leaf elements.
1. Fixed inability to delete AI-generated attributes when canceling AI-generated analytics.
