## Highlights

1. Enhanced AI Capabilities: Panel intelligent interpretation now includes a dialog box where users can ask follow-up questions based on AI insights; more nodes support root cause analysis; recommended questions and panels now directly invoke real-time element statistics for more accurate AI recommendations; AI Chat element access control to meet enterprise-level information security requirements
1. Advanced Analytics Enhancement: Trend charts now support Ad Hoc event windows, optimized display for multiple event window types, advanced analytics entry moved to action bar, new window configuration added to analysis detail pages, enhanced scatter plot analysis interface, etc.
1. Panel Optimization: Added multiple panel display configuration options such as color, decimal places, and UOM; improved display for Gauge, Bar Gauge, Stat and other elements; added data link options, new Zoom In/Out and drag-to-pan operations, etc.
1. Sample Data and Scenario Optimization: Sample data now more closely reflects real business scenarios, with pre-configured default panels, analytics, and events covering industry use cases
1. Data Management and System Performance Improvements: Added proactive index repair functionality, CSV export now includes success notifications, Telemetry data import validation, element management converted to asynchronous operations for improved performance, etc.
1. Canvas Supports Design Drawing Uploads: Added upload and rendering support for dxf, zip, svg, Visio diagrams, and CAD drawings (.dxf); design drawings can now be linked to element attributes for integrated display, enhancing canvas visualization.
1. Other Improvements and Bug Fixes
