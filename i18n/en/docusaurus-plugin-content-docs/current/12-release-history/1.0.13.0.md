1. enh: Add a monitoring task to the OPC import task to continuously monitor new table creation.

1. enh: Other panel types also display measurement units by default.
1. enh: Add configuration for display columns in the element template list page.
1. enh: Move the notification rules from the element's path bar to the action buttons of the event list.
1. enh: On the search results page, clicking the attribute name can redirect to the attribute details page.
1. enh: Display of multiple gauges in panels with Child Elements Grouping.
1. enh: The asset filtering of Excel Add-in can list the path of elements.
1. enh: OPC-UA import supports importing from multiple databases simultaneously.

1. enh: Improve the storage data for attribute data reference settting.
1. enh: Support panel insights during editing.
