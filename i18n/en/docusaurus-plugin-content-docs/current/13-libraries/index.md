# 13 Libraries

import DocCardList from '@theme/DocCardList';

**Libraries** is the central repository of reusable definitions used across TDengine IDMP. It is accessed from the **Libraries** menu in the main navigation bar.

The Libraries menu contains the following sections:

<table>
<colgroup><col style="width:11em"/><col/></colgroup>
<thead><tr><th>Section</th><th>Description</th></tr></thead>
<tbody>
<tr><td><strong>Element Template</strong></td><td>Define reusable element structures with attribute templates (see <a href="../03-data-modeling/01-elements.md">Chapter 3</a>)</td></tr>
<tr><td><strong>Event Template</strong></td><td>Define event schemas and notification rules (see <a href="../06-events/01-event-templates.md">Chapter 6</a>)</td></tr>
<tr><td><strong>Enumeration Sets</strong></td><td>Map integer values to human-readable labels</td></tr>
<tr><td><strong>Categories</strong></td><td>Define classification tags for elements, attributes, events, and more</td></tr>
<tr><td><strong>UOM</strong></td><td>Manage units of measurement and automatic conversion</td></tr>
<tr><td><strong>Composite Metric</strong></td><td>Define virtual metrics computed from other attributes (see <a href="../07-real-time-analysis/index.md">Chapter 7</a>)</td></tr>
</tbody>
</table>

This chapter covers Enumeration Sets, Categories, and UOM.
