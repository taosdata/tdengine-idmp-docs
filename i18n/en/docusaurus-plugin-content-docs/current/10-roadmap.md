# Product Roadmap

TDengine IDMP is under active development. The roadmap for 2026 is shown in the table below.

| Quarter | Features |
|---------|----------|
| 2026Q1 | <ol><li>Root cause analysis support</li><li>Manual input or CSV upload for collected data</li><li>Event statistics analysis panel</li><li>More panel types (Status History and Bar Gauge)</li><li>Data quality bit</li></ol> |
| 2026Q2 | <ol><li>Support for relational databases, including MySQL, PostgreSQL, Microsoft SQL Server, Oracle Database, and SAP HANA</li><li>Panel and dashboard drill-up and drill-down</li><li>Dashboard components and linked panels</li><li>Version control for configuration</li><li>Internal tables</li><li>Sub-events</li></ol> |
| 2026Q3 | <ol><li>Theme selection and customization for panels and dashboards</li><li>Data quality monitoring</li><li>Third-party chart plugins</li><li>More configurable options for panel legends, series, and axes</li><li>UI localization (French and Spanish)</li></ol> |
| 2026Q4 | <ol><li>System observability metrics</li><li>User groups support</li><li>Support for third-party time-series databases</li><li>Compatible with mobile clients</li></ol> |
