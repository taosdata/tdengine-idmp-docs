## Highlights

1. AI-powered Root Cause Analysis for Events: Quickly identify the most fundamental, underlying, and actionable root causes of quality anomalies.
1. Data Quality Bit Support: Add data quality bits for OPC-UA and CSV data, with support for data reference attributes and visualization analysis that includes data quality bits.
1. Panel & Visualization Improvements: Expanded library of chart templates，richer chart configuration options，gradient color controls for configuration panels，more chart types available in AI-recommended panels
1. Advanced Analytics Enhancements: Add more entry points for advanced analytics, scatter plots now support clustering and predictive analysis, optimized prediction capabilities for trend charts, envelope lines supported in event trend chart, support for time-normalized comparative analysis across multiple events
1. AI Trial Service: Extend AI free trial period from 3 days to 15 days.
1. Other Minor Improvements and Bug Fixes
