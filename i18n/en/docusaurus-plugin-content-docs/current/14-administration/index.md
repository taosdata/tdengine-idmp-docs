This chapter covers the administrative aspects of TDengine IDMP, including deployment architecture, capacity planning, user management, system configuration, and backup and recovery.

The Admin Console is accessed by clicking the avatar in the top-right corner and selecting **Admin Console**.

| Section | Description |
|---|---|
| **Deployment Architecture** | Overview of single-instance and high-availability deployment topologies |
| **Planning** | Hardware requirements, OS support, port assignments, and directory layout |
| **User Management** | Users, roles, permissions, and single sign-on (SSO) via OAuth 2.0 |
| **System Configuration** | Basic settings, notification contact points, notification templates, and email server |
| **Backup and Restore** | Configuring automated backups and restoring data from a backup |

:::note
**Monitoring and Logging** features are planned for a future release.
:::
