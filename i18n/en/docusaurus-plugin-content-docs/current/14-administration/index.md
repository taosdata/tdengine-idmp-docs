# 14 Administration

This chapter covers the account and system management aspects of TDengine IDMP, including personal settings, deployment architecture, capacity planning, installation, user management, system configuration, backup and recovery, and audit trail.

Personal settings are accessed from the top account item in the avatar menu. The **Admin Console** is accessed from the same menu.

| Section | Description |
|---|---|
| **Deployment Architecture** | Overview of single-instance and high-availability deployment topologies |
| **Planning** | Hardware requirements, OS support, port assignments, and directory layout |
| **Installation** | Detailed guides for Docker, installer package, Ansible, Helm, and other deployment methods. For Excel Add-In installation, see [Chapter 10](../10-excel-add-in/01-installation.md). |
| **User Management** | Users, roles, permissions, and single sign-on (SSO) via OAuth 2.0 |
| **Profile Settings and API Keys** | Manage your own profile, avatar, password, and user-level API keys |
| **System Configuration** | Basic settings, notification contact points, notification templates, and email server |
| **Backup and Restore** | Configuring automated backups and restoring data from a backup |
| **Audit Trail** | Tamper-proof operation logs with filtering, querying, and export for compliance and security traceability |
| **Sample Data** | Loading pre-built scenario packages to explore and demonstrate system features without a real data source |

:::note
**Monitoring and Logging** features are planned for a future release.
:::
