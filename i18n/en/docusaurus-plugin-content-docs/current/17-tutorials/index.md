# 17 Tutorials


The following tutorials walk through real-world IDMP deployments across common industrial scenarios. Each article is published on the TDengine website and covers end-to-end setup from data collection to visualization.

## TDengine in Action

- [Microgrid with Sparkplug B](https://tdengine.com/tdengine-idmp-in-action-microgrid-with-sparkplug-b/) — Connect a microgrid using the Sparkplug B protocol over MQTT, model the assets in IDMP, and build operational dashboards.

- [Industrial Boilers with Kepware](https://tdengine.com/tdengine-idmp-in-action-industrial-boilers-and-kepware/) — Integrate Kepware OPC-UA data into IDMP to monitor industrial boiler performance and configure alerting.

- [Electric Vehicles with MQTT](https://tdengine.com/tdengine-idmp-in-action-electric-vehicle-monitoring/) — Stream EV telemetry over MQTT into IDMP and create a live vehicle monitoring solution.

- [DC Monitoring with Telegraf](https://tdengine.com/tdengine-idmp-in-action-data-center-monitoring/) — Use Telegraf to collect data center metrics and feed them into IDMP for centralized monitoring and analysis.
