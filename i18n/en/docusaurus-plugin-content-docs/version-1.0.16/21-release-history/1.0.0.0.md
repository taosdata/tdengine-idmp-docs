## Features

1. Data modeling: IDMP manages data through a classic tree hierarchy. Each node in the tree corresponds to a sensor, device, production line, plant, or logical entity. IDMP refers to these nodes as elements; each element has its own attributes, real-time analytics, visualization panels, events, and more.
1. Data contextualization: For every element and each element's attributes, analytics, panels, and notifications, IDMP allows configuring descriptive information, categories, locations, physical units, limit values, and more — enriching data with business context and semantics.
1. Data standardization: Through templates for elements, attributes, analytics, panels, and events — along with attribute-to-database mappings, calculation expressions, and automatic unit conversion — IDMP provides data standardization capabilities that unify data from multiple heterogeneous sources.
1. Data visualization: IDMP includes a Grafana-style data visualization module with panels and dashboards. Supported chart types include trend charts, bar charts, pie charts, gauges, stat panels, rich text, tables, and more (scatter plots, maps, SCADA, and heatmaps planned for future releases).
1. Real-time analytics: Built on TDengine stream processing, IDMP provides real-time analytics. Trigger types include sliding windows, scheduled windows, event windows, state windows, and count windows. Calculations can be simple expressions or time-window and cross-device data aggregations.
1. Event management: Real-time analytics can trigger events; based on event severity and configuration, IDMP notifies the responsible personnel in real time. Generated events can be acknowledged and escalated. Users can easily view trend changes of related attributes at the time an event occurred to analyze root causes.
1. Proactive AI insights: IDMP uses AI large language models to intelligently perceive business scenarios based on collected data and proactively recommend panels and real-time analytics relevant to those scenarios. Users need not ask questions or possess deep business knowledge or IT skills to gain business insights.
1. AI-powered data Q&A: Users with strong business knowledge can ask IDMP to automatically create the visualization panels and real-time analytics they need, or ask any question related to system data. Because no IT skills are required, business users can obtain insights at any time.
1. Enterprise-grade services: IDMP provides role-based access control, data model version control (coming in a future release), single sign-on, data import/export, backup, and other features essential for enterprise operations.
