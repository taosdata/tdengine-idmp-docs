**Release Date: 2026-01-30**

## Features

1. Optimize the analytics creation and view pages.
1. Add an AI interpretation feature to the panel view page.
1. Support element and attribute categories in simple import and supertable-to-element mapping.
1. Panel advanced SQL supports the `element#name` substitution parameter.
1. Add the AI service to permission groups.
1. Events support reason codes.
1. Elements support being added to templates.
1. Add a new data import method: "Import from OPC."
1. Support specifying the display order of panels and dashboards and setting a default dashboard.
1. Panels support bar gauge charts.
1. Panel time windows support an interval offset.
1. Simple import and OPC import support selecting a parent element.

## Improvements

1. AI Q&A supports a maximum question count limit.
1. Remove forced scroll-to-bottom during AI Q&A streaming responses.
1. Excel Add-in supports saving cell settings.
1. Excel Add-in optimizes the time picker.
1. Improve the time range of panels generated by proactive AI insights.
1. Save the multi-swimlane option state to trend charts.
1. Add memory check to startup scripts.
1. Improve links included in AI Q&A answers.
1. Remove the UOM search and move it to the UOM list page.
1. Remove basic menu items from the element browser context menu.
1. Add end time display to the data import results page.
1. Add a "New Template" button to the dialog for adding elements to element templates.
1. Improve OPC-UA data import performance.
1. Display element paths in the element selector popup within panel and dashboard templates.
1. Each metric in multi-swimlane charts can independently auto-set its Y-axis max and min values.
1. Display element paths in search result and favorites list pages.
1. Display element paths in the top object popup on the search page.
1. Add an attribute selection button to delete attributes in trend charts.
1. Support saving multi-swimlane panels.
1. OPC import automatically saves the tag description as the element or attribute description.
1. Checkboxes on list pages support selecting all items across all pages.
1. Stat panels display units and last-updated timestamps.
1. Improve the progress dialog for data import.
1. Panel time picker supports time functions such as `today()`.
1. Multi-swimlane and related buttons are consolidated into the function action area at the top right of the panel.
1. Improve the Y-axis min/max calculation method for panels.
1. Multi-swimlane charts can be displayed in panel lists and dashboards.
