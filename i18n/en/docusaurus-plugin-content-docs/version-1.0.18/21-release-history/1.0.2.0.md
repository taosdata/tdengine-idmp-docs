**Release Date: 2025-08-22**

## Features

1. Attributes and attribute templates support formula and string value types.
1. TDengine connections support import from CSV files.
1. Add documentation for configuring notification channels.
1. Show error messages after sample loading failures.
1. Expressions support formula and string types and can be evaluated.
1. Show a progress bar while samples are loading.
1. Prevent users from directly adding data-reference attributes to elements created from element templates.
1. Support direct upgrade from version 1.0.0.0 and 1.0.1.x to 1.0.2.0.

## Improvements

1. The batch attribute value retrieval API now returns error details.
