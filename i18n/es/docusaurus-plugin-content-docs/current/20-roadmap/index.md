TDengine IDMP is under active development. The 2026 annual roadmap is shown below.

| Quarter | Planned Features |
|---|---|
| 2026 Q1 | <ol><li>AI-based root cause analysis for events</li><li>Manual data entry and CSV upload for collected data</li><li>Event statistics analysis panel</li><li>Additional panel types (State History and Bar Gauge)</li><li>Data quality bit support</li></ol> |
| 2026 Q2 | <ol><li>Panel and dashboard drill-up and drill-down navigation</li><li>Dashboard components and linked panels</li><li>Version control for configuration</li><li>Internal table support</li><li>Sub-event support</li></ol> |
| 2026 Q3 | <ol><li>Panel and dashboard theme selection and customization</li><li>Data quality monitoring</li><li>Third-party chart plugin support</li><li>More configurable options for panel legends, series, and axes</li><li>French and Spanish language support</li></ol> |
| 2026 Q4 | <ol><li>System observability metrics</li><li>User group support</li><li>Relational database integration</li><li>Third-party time-series database support</li><li>Mobile client</li></ol> |
