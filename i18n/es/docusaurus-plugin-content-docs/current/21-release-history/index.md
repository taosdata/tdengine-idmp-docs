TDengine IDMP uses a four-part version number in the format `MAJOR.MINOR.PATCH.HOTFIX`:

- **MAJOR:** A major architectural change. Direct upgrade is not supported — contact TDengine support for migration assistance.
- **MINOR:** A significant new feature release. Downgrade after upgrade is not supported.
- **PATCH:** New features or bug fixes. Downgrade after upgrade is supported.
- **Hotfix:** Emergency fix release. Downgrade after upgrade is supported.
