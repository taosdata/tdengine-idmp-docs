## Features

1. Support HTTPS.
1. CSV import supports filling in element description information.
1. Support Excel Data Link add-in. (Preview)
1. Data reference attributes support selecting system enum types.
1. Element search supports setting complex attribute conditions.
1. Element general attributes support uploading documents.
1. SCADA panels support preview, save, and edit.
1. Fixed child element list saved as a panel showing inconsistent attributes.

## Improvements

None.

## Bug Fixes

1. Fixed iframes failing to load the frontend.
1. Fixed AI connections not supporting self-signed certificates.
1. Fixed incorrect time-series data value calculations in license checks.
1. Fixed the connection context menu in the tree view failing to open the simple import page.
1. Fixed copying data reference attribute templates querying new attribute values for elements incorrectly.
1. Fixed AI recommended questions failing to recommend questions for multiple child element templates.
1. Fixed inability to view panels generated from recommended questions.
1. Fixed child element aggregation panels not returning results in dashboards.
1. Fixed IDMP cloud service reporting an error when logging out after login.
