## Nuevas funcionalidades

1. Los paneles admiten el uso de elementos como dimensiones.
1. Agregar soporte para estadísticas de métricas de paneles.
1. Las plantillas de elementos admiten herencia de múltiples plantillas base.
1. Agregar documentos de elementos como material de referencia para las preguntas recomendadas de paneles y análisis.

## Mejoras

1. Las tareas de importación OPC agregan una tarea de escucha para monitorear continuamente la creación de nuevas tablas.
1. Cambiar la visualización de unidades de medida del eje Y del panel a forma abreviada.
1. Otros tipos de paneles también muestran unidades de medida por defecto.
1. Agregar configuración de visualización de columnas a la página de lista de plantillas de elementos.
1. Mover las reglas de notificación de la barra de ruta del elemento a los botones de acción en la lista de eventos.
1. Hacer clic en el nombre de un atributo en los resultados de búsqueda de atributos navega a la página de detalles del atributo.
1. Mejorar la visualización de múltiples paneles de control en los paneles de agregación de elementos secundarios.
1. El filtro de activos del complemento Excel Add-in puede listar rutas de elementos.
1. La importación OPC-UA admite importar desde múltiples bases de datos simultáneamente.
1. Mejorar múltiples interacciones de usuario para la eliminación de conjuntos de enumeraciones.
1. Mejorar la estructura de almacenamiento de referencias de datos de atributos.
1. Soporte para interpretación por IA mientras se editan paneles.
1. Las listas de plantillas de atributos de las plantillas de elementos pueden mostrar plantillas de atributos de las plantillas base.
1. Optimizar la estructura de almacenamiento de atributos y eliminar las partes duplicadas de las plantillas de atributos.
1. Agregar APIs de eliminación masiva para listas de plantillas de elementos y eventos en el backend.
1. Los análisis pueden gestionarse directamente desde el backend de administración.

1. Corregido que la copia de análisis produjera nombres de cálculo de flujo duplicados.
1. Corregido que no hubiera respuesta al cambiar el inicio de sesión entre usuarios con diferentes permisos.
1. Corregidas excepciones durante la importación OPC-UA.
