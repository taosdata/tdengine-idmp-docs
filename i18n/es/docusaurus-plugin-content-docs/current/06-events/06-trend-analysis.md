# 6.6 Análisis de Tendencias

Cuando ocurre un evento, entender el comportamiento de los datos alrededor de ese evento es esencial para la investigación. TDengine IDMP proporciona un camino directo desde cualquier evento hacia un gráfico de tendencias preconfigurado con el intervalo de tiempo del evento.

## 6.6.1 Abrir un Gráfico de Tendencias desde un Evento

Puede abrir un gráfico de tendencias para cualquier evento de dos maneras:

**Desde la lista de eventos:**

Pase el cursor sobre una fila de evento para mostrar el menú **⋮**, luego haga clic en **Análisis de Tendencias**. IDMP abre el gráfico de tendencias del elemento asociado, ampliado automáticamente a la hora de inicio y fin del evento.

**Desde la página de detalle del evento:**

En la pestaña General, haga clic en el icono **Análisis de Tendencias** en la barra de herramientas para abrir la misma vista.

## 6.6.2 Trabajar con el Gráfico de Tendencias del Evento

El gráfico de tendencias abierto desde un evento muestra los atributos de series temporales del elemento durante la ventana de tiempo del evento, con las horas de inicio y fin del evento marcadas en el gráfico.

Se pueden añadir múltiples eventos al mismo análisis de tendencias, lo que permite superponer y comparar diferentes eventos uno junto al otro — por ejemplo, comparar el comportamiento del sensor durante dos ocurrencias de fallos separadas.

Desde la vista del gráfico puede añadir atributos adicionales del elemento o de otros elementos en el árbol de activos, y ampliar o desplazar para ajustar la ventana de tiempo.

:::note
El Análisis de Tendencias es una función potente con muchas capacidades. Una sección dedicada en el capítulo de **Análisis Avanzado** lo cubre con todo detalle.
:::
