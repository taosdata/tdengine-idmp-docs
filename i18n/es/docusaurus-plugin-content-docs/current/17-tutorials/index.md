# 17. Tutoriales

- [TDengine IDMP impulsa la "cuádruple modernización" en la fabricación de cigarrillos (con instrucciones paso a paso)](https://www.taosdata.com/idmp-application-scenarios/33903.html) — Prácticas de gestión digital de datos de procesos en la fabricación de cigarrillos
- [TDengine IDMP protege las operaciones inteligentes de redes de gas y petróleo (con instrucciones paso a paso)](https://www.taosdata.com/idmp-application-scenarios/33632.html) — Monitoreo en tiempo real y análisis de datos operativos de redes de gas y petróleo

- [TDengine IDMP transforma las operaciones inteligentes de servicios de agua (con instrucciones paso a paso)](https://www.taosdata.com/idmp-application-scenarios/33626.html) — Práctica de gestión digital integral de datos operativos de servicios de agua
