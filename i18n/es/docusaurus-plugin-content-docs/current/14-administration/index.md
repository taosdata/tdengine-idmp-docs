# 14. Administración

Este capítulo cubre los aspectos administrativos de TDengine IDMP, incluyendo la arquitectura de despliegue, la planificación de capacidad, la gestión de usuarios, la configuración del sistema y la copia de seguridad y restauración.

La consola de administración se accede haciendo clic en el avatar en la esquina superior derecha y seleccionando **Consola de administración**.

| Sección | Descripción |
|---|---|
| **Arquitectura de despliegue** | Descripción general de las topologías de despliegue de instancia única y alta disponibilidad |
| **Planificación** | Requisitos de hardware, soporte de SO, asignación de puertos y estructura de directorios |
| **Gestión de usuarios** | Usuarios, roles, permisos e inicio de sesión único (SSO) mediante OAuth 2.0 |
| **Configuración del sistema** | Ajustes básicos, puntos de contacto de notificaciones, plantillas de notificaciones y servidor de correo electrónico |
| **Copia de seguridad y restaurar** | Configuración de copias de seguridad automáticas y restauración de datos desde una copia de seguridad |
| **Datos de muestra** | Carga de paquetes de escenarios preconfigurados para explorar y demostrar las funciones del sistema sin una fuente de datos real |

:::note
Las funciones de **Monitoreo y registro** están planificadas para una versión futura.
:::
