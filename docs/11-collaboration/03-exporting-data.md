# 11.3 导出数据

在 IDMP 中，任何表格或列表视图都可以导出为 CSV 文件。这使您无需编写任何查询或使用 Excel 插件，即可方便地将数据从 IDMP 导出，用于电子表格、报告或其他工具。

## CSV 导出可用位置

CSV 导出选项在 IDMP 的每个表格视图中均可使用，包括：

| 视图 | 导出内容 |
|---|---|
| **元素浏览器 — 子元素列表** | 所选元素下的子元素列表 |
| **元素浏览器 — 属性列表** | 所选元素的属性值和元数据 |
| **事件列表** | 包含时间戳、严重级别和状态的已过滤事件记录 |
| **事件过滤结果** | 符合事件搜索条件的事件 |
| **属性过滤结果** | 符合搜索条件的属性元数据 |
| **资产过滤结果** | 符合搜索条件的元素元数据 |
| **分析结果** | 已保存分析查询的输出行 |

## 如何导出

在表格上方的工具栏中查找**导出**或**下载**图标（通常为向下箭头或 CSV 图标）。点击该图标可将当前显示的数据（包括所有已激活的过滤器）下载为 CSV 文件。

导出内容反映的是下载时表格中可见的数据。请先应用过滤器缩小数据范围，再进行导出。

## Excel 插件

如需更结构化或定期将数据检索到 Excel 中，[Excel 插件](../10-excel-add-in/index.md)提供了专门构建的集成方案，让您能够直接将实时或历史时序数据拉取到工作表单元格中，并对时间范围、聚合方式和输出布局进行完整控制。
