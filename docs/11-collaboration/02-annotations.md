# 11.2 注解

注解允许您在元素、事件以及（未来版本中的）面板和仪表板上附加自由文本备注。它们作为共享日志——对所有有访问权限的人可见——用于记录观察结果、纠正措施、调查发现或任何对其他团队成员有用的上下文信息。

## 为元素添加注解

1. 在元素浏览器中，选择一个元素并点击**通用**选项卡。
2. 向下滚动到**注解**可折叠区域并展开它。
3. 点击 **+ 新建注解**。
4. 在弹出的对话框中输入您的备注，然后点击**确认**。

注解保存后将显示在"注解"区域下的列表中，显示备注文本、作者和时间戳。

## 为事件添加注解

1. 打开一个事件进入其详情页，然后点击**通用**选项卡。
2. 向下滚动到**注解**可折叠区域并展开它。
3. 点击 **+ 新建注解**。
4. 输入您的备注并点击**确认**。

事件注解适用于记录调查内容、采取的措施或确认事件的原因。它们通过在状态变更旁附加叙述性说明，对确认状态起到补充作用。

## 为面板和仪表板添加注解

:::note
面板和仪表板注解计划在未来版本中推出。
:::

在即将发布的版本中，面板和仪表板将支持通过右侧专用控制图标添加注解。点击该图标将打开一个注解面板，其工作方式与元素注解相同——允许您添加带时间戳的备注，对查看该面板或仪表板的所有人可见。
