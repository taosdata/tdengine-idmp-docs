# 11.1 共享仪表板与面板

IDMP 让您能够方便地与同事共享可视化内容，无需他们重新配置相同的设置。仪表板和单个面板都可以直接从可视化界面进行共享。

## 共享仪表板

仪表板可以共享，让其他用户打开并查看相同的面板布局、时间范围和配置。要共享仪表板，请打开它并使用仪表板工具栏中的**分享**选项。IDMP 会生成一个可共享链接，或允许您复制仪表板配置。

您也可以通过 iframe 标签将仪表板集成到您的系统页面中：

```html
<iframe
  src="集成链接"
  width="100%"
  height="600"
  frameborder="0"
  sandbox="allow-same-origin allow-scripts"
></iframe>
```

出于安全考虑，建议在集成页面上配置合适的 Content Security Policy（CSP）头，并通过 `sandbox` 属性限制 iframe 的能力。

也可以通过链接在新页面中打开仪表板：

```html
<a href="集成链接" target="_blank">打开仪表板</a>
```

完整的共享工作流程请参阅[仪表板](../04-visualization/04-dashboards.md)。

## 共享面板

单个面板可以独立于其父仪表板进行共享或导出。打开面板菜单（面板标题上的三点图标）并选择共享选项。

面板级共享选项请参阅[面板](../04-visualization/01-panels.md)。

## 访问控制

共享内容受 IDMP 基于角色的访问控制约束。接收者必须对底层元素和属性至少具有读取权限，才能在共享的仪表板或面板中查看实时数据。
