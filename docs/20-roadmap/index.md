# 20. 产品路线图

本路线图介绍 TDengine IDMP 未来的产品发展方向，帮助用户规划长期使用与集成方案。

路线图中的功能按交付节奏分为三类：

- **Now（进行中）**：当前正在开发、即将在近期版本中陆续发布的能力。
- **Next（规划中）**：已列入计划、将在 Now 之后启动的研发方向。
- **Later（探索中）**：更远期的战略方向，具体形态仍在调研与打磨中。

> 本路线图仅反映当前的研发计划与方向。我们会根据客户反馈和市场变化持续调整，具体功能的发布时间与范围请以正式发布说明为准。

---

## 产品主题

- **分析与可视化**：不断扩展图表类型、分析算法与交互能力，帮助用户从工业数据中获取更深层的洞察。
- **企业级就绪**：持续完善安全、合规、可用性与可维护性，满足大型企业生产级部署的要求。
- **平台基础能力**：强化数据建模、表达式、事件、限值、数据管理等底座，让平台更完整、更易用。
- **AI 原生体验**：将 AI 深度融入平台的每一个场景，让 AI 成为每位用户操作 IDMP 的智能助手。

## Now — 进行中

> 近期版本中陆续发布的能力。

### 分析与可视化

- **更多图表类型**：新增 K 线图、热力图、直方图。K 线图适合展示指标波动区间，热力图直观呈现大规模设备的运行状态分布，直方图帮助了解数据的统计分布特征。
- **组态面板增强**：支持组态模板化、面板引用与数据最后更新时间展示。模板化让组态监控可快速复制到同类设备，减少重复配置。
- **面板与仪表板交互优化**：支持上钻、下钻与联动，让用户可以沿着业务层级逐步深入分析。
- **Excel Add-in 增强**：让习惯使用 Excel 做分析的用户更便捷地访问 IDMP 中的数据。
- **相似度分析（Profile Search）**：在历史数据中搜索与目标曲线形态相似的时间段，辅助异常工况根因分析与优质批次工艺参数总结。
- **事件关联分析（Association Analysis）**：从历史事件中挖掘"哪些事件经常一起发生"，通过量化指标衡量关联强度，为预测性维护提供基础。
- **质量分析应用**：统一的质量分析入口，整合 SPC 监控、过程能力分析（Cp/Cpk）、批次比较与质量因素探索，品控团队可在一个界面内完成从监控、告警到分析的全流程。
- **模型开发与管理**：在平台内完成工业分析模型的全生命周期管理，涵盖开发、训练、发布、部署、监控与重训练，首期支持时序预测、异常检测、聚类、分类、回归五类场景。工艺工程师与业务分析师无需编写代码即可构建模型应用。

### 企业级就绪

- **用户与权限管理升级**：用户组权限、API Key / Token 管理、密码与账号策略、批量用户导入，满足大规模部署下的用户管理需求。
- **企业目录集成与单点登录（SSO）**：支持 LDAP / AD，企业员工使用已有公司账号即可登录 IDMP，无需单独管理密码。
- **多因素认证（MFA / 2FA）**：为登录和敏感操作提供额外的身份验证保障。
- **审计日志**：记录所有关键操作，支持筛选查询与导出，满足合规审计与安全追溯要求。
- **版本控制与审批流程**：基于 Git 为元素配置、模板、事件、模型等关键对象提供变更管理、评审与回滚能力，满足制药、食品、能源等受监管行业对变更可控、可追溯的合规要求。
- **高可用集群部署**：多实例负载均衡与故障自动切换，确保单节点故障时业务不中断，提升系统可靠性。
- **系统备份与恢复**：支持定时备份、热备与加密存储，快速恢复系统至指定时点。

### 平台基础能力

- **限值（Limits）管理**：统一管理元素属性的安全边界与质量规格上下限（USL/LSL），一次定义、全平台引用。修改限值时，所有相关的报警规则、可视化面板与计算公式自动保持一致，避免散落在多个系统中的标准不一致问题。
- **跨元素属性引用**：表达式引擎突破单一元素边界，可直接引用上下游设备的属性参与计算，支持更复杂的跨设备联动分析。
- **复杂事件处理（子事件）**：父子多级事件结构，支持多条件、多阶段、跨设备的复合事件。打开任意事件即可看到完整的子事件构成与时间线，让复合故障一目了然。
- **内部表**：在平台内直接管理关系型业务数据，与时序数据联合使用，避免跨系统查询的复杂性，让数据分析与报表更完整。
- **数据导入增强**：提供更灵活的手动输入与 CSV 上传，简化历史数据与业务数据的接入。
- **数据采集端状态监控**：实时掌握采集网关与设备的健康状态，第一时间发现数据中断或采集异常，保障数据的完整性与及时性。
- **业务使用量统计**：展示用户数、元素数、API 调用量、存储用量等关键指标，为容量规划与投资回报评估提供数据支撑。

### AI 原生体验

- **Agent Loop 架构升级**：将 IDMP AI 技术架构升级为 Agent Loop，显著提升复杂任务的意图识别与执行能力。
- **IDMP CLI 命令行**：提供脚本化操作 IDMP 的能力，便于与企业现有运维和数据流水线集成。
- **IDMP Skills 管理**：用户可自行管理和扩展 AI 能力，按自己的业务场景定制智能助手。
- **AI 配置管理界面**：集中管理 AI 模型、Skills 与提示词等配置，简化 AI 功能的运维。

## Next — 规划中

> 在 Now 之后启动的研发方向。

### 分析与可视化

- **可视化主题与样式定制**：面板与仪表板支持主题选择与更精细的样式定制。
- **第三方图形插件**：开放图形插件机制，允许集成第三方可视化组件。
- **面板配置项扩展**：图例、系列、坐标轴等提供更丰富的可配置项。

### 企业级就绪

- **系统可观测性**：对外暴露系统级可观测指标，便于纳入企业统一监控体系。

### 平台基础能力

- **数据质量监测**：提供完整的数据质量监测能力，量化数据的完整性、及时性与有效性。
- **关系库支持**：在内部表基础上进一步扩展对关系型数据库的管理能力。
- **第三方时序库对接**：支持将其他时序数据库作为数据源接入 IDMP。

### 全球化

- **更多语言支持**：增加语种支持，扩展全球化覆盖。

- **移动端访问**：提供手机客户端，支持随时随地查看监控、事件与分析。
