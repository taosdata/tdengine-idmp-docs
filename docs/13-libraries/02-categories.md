# 13.2 分类

**分类**是可应用于 IDMP 中各类对象的分类标签，便于过滤、搜索和整理。任何元素、属性、事件、仪表板、面板或分析都可以打上一个或多个分类标签。

分类在**基础库 → 分类**中管理。

## 13.2.1 分类列表

列表显示所有已定义的分类，包含以下列：

<table>
<colgroup><col style="width:5em"/><col/></colgroup>
<thead><tr><th>列</th><th>说明</th></tr></thead>
<tbody>
<tr><td><strong>名称</strong></td><td>分类名称</td></tr>
<tr><td><strong>描述</strong></td><td>可选描述</td></tr>
</tbody>
</table>

每个分类还有一个**类型**，决定它可以应用于哪种对象。列表按对象类型组织。点击分类名称可编辑。使用行上的 **⋮** 菜单可编辑或删除。

## 13.2.2 创建分类

点击 **+** 创建新分类。填写以下字段：

<table>
<colgroup><col style="width:9em"/><col/></colgroup>
<thead><tr><th>字段</th><th>说明</th></tr></thead>
<tbody>
<tr><td><strong>名称</strong>（必填）</td><td>在其类型范围内的唯一名称。支持字母、数字、下划线、连字符和空格。</td></tr>
<tr><td><strong>类型</strong>（必填）</td><td>此分类适用的对象类型：<code>元素</code>、<code>属性</code>、<code>分析</code>、<code>仪表板</code>、<code>面板</code>或<code>事件</code>。</td></tr>
<tr><td><strong>描述</strong></td><td>可选描述。</td></tr>
</tbody>
</table>

点击**保存**创建分类。

## 13.2.3 使用分类

在创建或编辑对象（元素、属性、事件、面板、仪表板或分析）时，您可以在**分类**字段中为其分配一个或多个分类。分类必须与对象类型匹配——例如，编辑元素时只显示类型为**元素**的分类。

分配后，分类将作为过滤选项出现在整个 IDMP 中：

- 在**元素浏览器**中，使用分类过滤器将资产树缩小到特定分类的元素。
- 在**仪表板**和**事件**列表中，按分类过滤可快速找到相关面板或事件。
- 在**基础库**中，分类有助于整理大量元素模板和事件模板。

## 13.2.4 编辑与删除

要编辑分类，点击其名称。要删除，使用 **⋮** 菜单。删除分类不会影响已分配该分类的对象——分类标签只是从这些对象上移除。
