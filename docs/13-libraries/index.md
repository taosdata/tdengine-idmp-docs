# 13. 基础库

**基础库**是 TDengine IDMP 中可复用定义的中央存储库，可在整个平台中共享使用。通过主导航栏中的**基础库**菜单访问。

基础库菜单包含以下部分：

| 部分 | 说明 |
|---|---|
| **元素模板** | 定义带有属性模板的可复用元素结构（参见[第 3 章](../03-data-modeling/01-elements.md)） |
| **事件模板** | 定义事件模式和通知规则（参见[第 6 章](../06-events/02-event-templates.md)） |
| **枚举集** | 将整数值映射为可读标签 |
| **分类** | 为元素、属性、事件等定义分类标签 |
| **计量单位** | 管理计量单位及自动转换 |
| **复合指标** | 定义由其他属性计算得出的虚拟指标（参见[第 7 章](../07-real-time-analysis/index.md)） |

本章介绍枚举集、分类和计量单位。
