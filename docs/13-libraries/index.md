# 13. 基础库

**基础库**是 TDengine IDMP 中可复用定义的中央存储库，可在整个平台中共享使用。通过主导航栏中的**基础库**菜单访问。

基础库菜单包含以下部分：

<table>
<colgroup><col style="width:7em"/><col/></colgroup>
<thead><tr><th>部分</th><th>说明</th></tr></thead>
<tbody>
<tr><td><strong>元素模板</strong></td><td>定义带有属性模板的可复用元素结构（参见<a href="../03-data-modeling/01-elements.md">第 3 章</a>）</td></tr>
<tr><td><strong>事件模板</strong></td><td>定义事件模式和通知规则（参见<a href="../06-events/01-event-templates.md">第 6 章</a>）</td></tr>
<tr><td><strong>枚举集</strong></td><td>将整数值映射为可读标签</td></tr>
<tr><td><strong>分类</strong></td><td>为元素、属性、事件等定义分类标签</td></tr>
<tr><td><strong>计量单位</strong></td><td>管理计量单位及自动转换</td></tr>
<tr><td><strong>复合指标</strong></td><td>定义由其他属性计算得出的虚拟指标（参见<a href="../07-real-time-analysis/index.md">第 7 章</a>）</td></tr>
</tbody>
</table>

本章介绍枚举集、分类和计量单位。
