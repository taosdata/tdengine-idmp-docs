# 仪表板与面板通知

仪表板和面板可以定期发送快照通知给指定用户。

## 配置通知规则

在编辑面板或者仪表盘时，可以配置通知规则。

在编辑页面右侧的“通知规则”区域，点击“添加通知规则”按钮后，您可以进行以下配置：

- **通知频率**：选择发送通知的周期。
- **开始执行时间**：设置首次发送通知的时间。
- **结束时间**：设置通知规则的结束时间（可选）。
- **通知途径**：选择接收通知的方式。

如果选择周期性通知，后续的执行时间将根据开始执行时间进行计算。例如，若通知频率为每周一次，开始时间为周一上午9点，则通知将在每周一上午9点发送。结束时间为可选项，若不设置，则通知规则会一直生效。

通知规则配置完成后，点击保存按钮保存面板或仪表盘。

## 发送通知

当到达设置的通知时间时，系统会按照通知规则自动发送通知。如果是面板模板通知，那么每一个使用该模板的元素都会生成面板或仪表板快照并发送通知。
