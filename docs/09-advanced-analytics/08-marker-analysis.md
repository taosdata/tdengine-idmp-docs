# 9.8 标记线分析

标记线分析是分析面板提供的交互式数据比较工具。用户在分析面板中放置两条贯穿所有泳道的垂直标记线（Marker Line），随后移动标记线至图表上的任意位置，系统即可实时显示两条标记线位置之间的**时间差**以及**各属性值的差值**。

这是工业数据分析软件（Historian）中的一项标准诊断功能。在工业场景排查问题时，工程师往往需要快速比较两个时间点上多个属性的差异——例如比较故障时刻与正常运行时刻的温度、压力、转速等参数。标记线分析让这一操作无需导出数据或切换工具，在图表上即可完成。

## 9.8.1 功能原理

标记线分析的核心思路是：**在时间轴上固定两个参考点（标记线），然后通过移动标记线来动态比较任意两个时间点之间的属性差异。**

具体流程如下：

1. 用户在分析面板的查看模式下，点击操作栏中的**启用标记线**按钮。
2. 系统在分析面板的中心位置生成两条贯穿所有泳道的垂直红色标记线。
3. 用户将光标挪至任一标志线顶端，此时可以将标记线拖拽至感兴趣的时间位置（例如一个异常峰值、一次工况切换发生的时刻）。
4. 第二条标志线附近的提示框（Tooltip）中将实时显示：
   - **时间差（ΔT）：** 两条标记线所在位置的时间间隔
   - **各属性差值（ΔValue）：** 分析面板中每个属性在两条标记线位置的数值之差
5. 用户再次点击**标记线**按钮，两条标记线消失，恢复到普通浏览模式。

标记线分析不涉及对数据的修改写入操作，它仅是一种交互式可视化辅助功能，帮助用户在浏览历史数据时快速量化两个时间点之间的各项指标差异。

## 9.8.2 适用场景

标记线分析在工业领域具有广泛的应用价值，典型场景包括：

- **故障前后对比：** 将标记线放置在故障发生时刻，移动光标到故障前的正常运行区间，快速读取各传感器参数在故障前后的变化量
- **工况切换分析：** 将标记线放置在工况切换点（如负载变化、配方切换），移动光标到切换后的稳定区间，量化切换前后各属性的偏移
- **批次间差异比较：** 将标记线放置在某批次的特征时间点（如升温完成时刻），移动光标到另一批次的对应位置，比较两个批次在关键节点上的参数差异
- **趋势斜率估算：** 将标记线放置在一段单调趋势的起点，移动光标到终点，通过 ΔValue 和 ΔT 快速估算属性的变化速率
- **事件响应时间测量：** 将标记线放置在输入信号发生变化的时刻，移动光标到输出信号开始响应的位置，直接读取响应延迟时间

## 9.8.3 使用入口

在**分析面板**（Analysis Chart）的查看模式下，点击操作栏中的**标记线**（Marker Line）按钮即可使用。

### 操作步骤

1. 打开或创建一个**分析面板**，添加需要分析的时序属性。
2. 点击操作栏中的**标记线**按钮。
3. 分析面板中心位置出现两条贯穿所有泳道的垂直红色标记线。
4. 拖拽任一标记线到目标时间位置（例如故障发生时刻、工况切换点）。
5. 第二条标记线的提示框中将实时显示 ΔT 和各属性的 ΔValue。
6. 分析完成后，用户再次点击**标记线**按钮，两条标记线消失。

### 提示框信息

当标记线处于激活状态时，提示框中实时显示以下信息：

| 信息项 | 说明 |
|---|---|
| **时间差（ΔT）** | 两条标记线的时间差值，格式根据时间跨度自适应（如 "5.5h"） |
| **各属性差值（ΔValue）** | 分析面板中每个属性在两条标记线之间的插值，绿色为正值，红色为负值 |
