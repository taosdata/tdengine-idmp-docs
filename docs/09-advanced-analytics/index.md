# 9. 高级分析

TDengine IDMP 的分析能力贯穿于整个平台。除了简单的可视化分析之外，IDMP 也提供了一整套面向工业时序数据的高级分析能力，包括时序预测、异常检测、缺失数据填补、聚类、回归、相关分析、批次分析、统计检验、关联规则、机器学习、模型管理等。

这些高级能力可帮助工业用户深入理解设备运行规律，发现数据中的隐藏模式，并基于数据做出更准确的决策。我们不仅在持续丰富高级分析能力，也计划针对具体工业场景推出开箱即用的分析模板与应用。

TDengine IDMP 的高级分析能力来源于内置的时序 AI 分析引擎 TDgpt。TDgpt 能够将时序数据分析服务无缝集成在 TDengine TSDB 的查询执行流程，直接对时序数据执行计算密集型分析任务，并通过 `FORECAST()`、`ANOMALY_WINDOW()` 等 SQL 函数对外提供高级分析能力。

在设计这些高级分析功能时，我们并没有将其构建为独立的应用，而是直接嵌入每一个具体分析面板与页面之中，让用户在当前分析场景下方便快捷地使用这些高级分析能力。

本章将分散在 IDMP 平台中的各种高级分析能力进行汇总阐述。
