# 9. 高级分析

TDengine IDMP 的分析能力贯穿于整个平台。除了基础的可视化分析之外，IDMP 还提供了一整套面向工业时序数据的高级分析能力，包括时序预测、异常检测、缺失数据填补、聚类、回归、相关分析、批次事件分析、关联规则、机器学习、模型管理等。

这些高级能力可帮助工业用户深入理解设备运行规律，发现数据中的隐藏模式，并基于数据做出更准确的决策。我们不仅在持续丰富高级分析能力，也计划针对具体工业场景推出开箱即用的分析模板与应用。

TDengine IDMP 的高级分析能力来源于内置的时序 AI 分析引擎 TDgpt。TDgpt 将时序数据分析服务无缝集成在 TDengine TSDB 的查询执行流程中，直接对时序数据执行计算密集型分析任务，并通过 `FORECAST()`、`ANOMALY_WINDOW()` 等 SQL 函数对外提供高级分析能力。

这些高级分析功能并非以独立应用的形式提供，而是直接嵌入每一个具体的分析面板与页面之中——让用户无需切换上下文，即可在当前分析场景下随时调用。

本章将分散在 IDMP 平台中的各种高级分析能力进行汇总阐述。
