## 加载示例场景

为了帮助您快速上手，TDengine IDMP 内置了以下示例场景，您可以通过加载这些示例数据来体验系统的强大功能。

:::important
每次只能加载一个示例场景。请等待当前场景加载完毕后，再加载下一个场景。
:::

| 场景 | 说明 |
|---|---|
| 公共事业 | 智能电表、智能水表实时数据监测，支持精细化能源管理与异常检测 |
| 卷烟制丝 | 卷烟厂制丝工艺监控 |
| 车辆管理 | 物流企业车辆实时位置、速度、路线及超速告警追踪管理 |
| 光伏发电 | 光伏电站逆变器及环境传感器实时数据监测 |
| 油井生产 | 油田油井产量、压力及含水率实时监测 |
| 污水处理 | 污水处理厂生物反应器、膜池及进出水水质传感器实时数据监控 |
| 新能源集控 | 风光储多能源数据汇聚，支持区域级统一监控与智能调度 |

:::tip
以下示例将选用**公共事业**场景的数据进行演示：请选中"公共事业"场景，然后点击**确认**，系统会自动加载该场景的数据。加载过程可能需要几分钟，请耐心等待。
:::

## 查看元素信息

加载示例场景后，您将进入 TDengine IDMP 的主 Web 界面，默认来到的是**元素浏览器**页面。

1. 在侧边栏中选择**元素**，当"公共事业"场景加载完成后，即可查看其下具体的元素信息。
2. 在"公共事业"场景下，包含北京、河北、河南及其地级市或区县的电表、水表信息，您可以点击左侧的树形结构浏览不同的地理位置。
3. 选择 **北京 → 朝阳 → 三元桥街道 → em-10** 这个元素，其代表北京市朝阳区三元桥街道编号为 10 的电表。
4. 在顶部面包屑导航栏的下拉列表中，选择**通用**，查看其描述等通用信息。
5. 切换至**属性**，查看电流、电压等物理量属性。

![查看元素属性](/docs-img/get-started/attributes.png)

## 体验 AI 生成面板

在 TDengine IDMP 中，最强大的功能之一是通过 AI 进行数据面板的自动生成。

1. 选择 **em-10** 元素，点击顶部面包屑导航栏中的下拉列表，选择**面板**。
2. 如果**AI 推荐**已开启，即可在"面板"页面查看到 AI 推荐的数据面板，点击 **+ 更多推荐**，系统会自动生成更多的面板供您选择。
3. 您还可以在推荐面板下方的对话框中，以自然语言的方式告诉 AI 您想要的面板，例如：

   *"智能电表 em-10 过去 24 小时每分钟电压和电流的变化，以折线图的方式展示。"*

   然后点击**问我一下**，AI 会自动生成您期望的面板。

![AI 生成面板](/docs-img/get-started/panels.png)

## 体验 AI 分析

AI 分析功能可以帮助您实时感知数据的变化，当数据发生异常时，系统会自动产生相关的事件，并触发告警通知。

1. 选择 **em-10** 元素，点击顶部面包屑导航栏中的下拉列表，选择**分析**。
2. 如果**AI 推荐**已开启，即可在"分析"页面查看到 AI 推荐的问题，点击推荐问题右侧的**换一换**，系统会自动更新推荐的问题。
3. 点击感兴趣的问题链接，进入分析创建页面，在这里可以对 AI 创建的分析进行进一步的配置和调整，确认后，点击**保存**。
4. 您还可以在推荐问题旁的文本框中，以自然语言告诉 AI 您想要分析的问题，例如：

   *"智能电表 em-10 的功率波动超过 ±20% 持续 30 分钟，产生"警告"级别的告警，并计算波动幅度。"*

   然后点击回车，AI 就会自动生成您期望的分析。
