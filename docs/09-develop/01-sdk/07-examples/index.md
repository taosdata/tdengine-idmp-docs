# 示例

本章提供面向真实业务场景的完整代码示例。每个示例均可独立运行，您可以直接复制并根据实际环境修改。

### 运行前准备

所有示例均从以下环境变量读取配置：

```bash
export IDMP_HOST=http://your-idmp-server:6042
export IDMP_USERNAME=your_username
export IDMP_PASSWORD=your_password
# 或者，如果您已有 Token：
export BEARER_TOKEN=your_token
```
