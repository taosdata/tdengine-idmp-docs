# 4.6 定时报告

定时报告功能可按照周期计划，将面板或仪表板的渲染图片发送给一个或多个接收人。在配置的时间点，IDMP 捕获面板或仪表板的当前状态，生成 PNG 图片，并通过配置的通知联系点发送——无需登录或手动导出。

常见用途包括：每周发送给管理层的能源汇总仪表板、每日发送给班组工程师的生产线趋势图，以及按需触发的一次性面板导出。

## 4.6.1 为面板配置定时报告

定时报告交付在面板编辑器的**通知规则**部分进行配置。

1. 打开面板编辑器（在任意面板卡片或查看模式中点击**编辑**）。
2. 在右侧设置面板中，滚动到**通知规则**，点击 **+** 添加新规则。
3. 配置规则字段：

<table>
<colgroup><col style="width:9em"/><col/></colgroup>
<thead><tr><th>字段</th><th>说明</th></tr></thead>
<tbody>
<tr><td><strong>频率</strong></td><td>发送报告的频率：<strong>单次</strong>（仅一次）、<strong>每天</strong>、<strong>每周</strong>或<strong>每月</strong></td></tr>
<tr><td><strong>任务开始时间</strong></td><td>首次（或唯一一次）交付的日期和时间。必填。</td></tr>
<tr><td><strong>结束日期</strong></td><td>周期计划停止的日期。可选——如留空，计划无限期运行。</td></tr>
<tr><td><strong>通知联系点</strong></td><td>接收报告的通知渠道。从下拉列表中选择一个或多个预配置的联系点。必填。</td></tr>
</tbody>
</table>

4. 再次点击 **+** 可添加具有不同频率或接收人的附加规则。
5. 点击**保存**保存面板并激活计划。

每条已保存的规则以一行显示在通知规则部分。点击行上的删除图标可删除规则。展开折叠的规则行可编辑其设置。

## 4.6.2 为仪表板配置定时报告

仪表板级报告会发送完整仪表板的图片，而不是单个面板。配置方法与面板级规则相同。

1. 打开仪表板编辑器。
2. 在右侧设置面板中，找到**通知规则**字段。
3. 点击 **+** 添加规则，填写相同的字段：频率、任务开始时间、结束日期和通知联系点。
4. 点击**保存**激活计划。

触发时，IDMP 渲染完整仪表板画布——仪表板默认时间范围下的所有面板——并以图片形式发送。

## 4.6.3 交付工作原理

定时报告触发时：

- IDMP 按照面板或仪表板在该时刻的显示效果进行渲染，使用面板或仪表板保存的时间范围配置。
- 报告生成为 PNG 图片。
- 图片被发送到该规则配置的所有联系点。
- 如果面板属于某个模板，则为使用该模板的每个元素分别生成并发送报告。

单次频率规则在任务开始时间触发一次后即失效。周期规则（每天、每周、每月）按照相对于任务开始时间的相同计划触发——例如，周一 09:00 开始的每周规则每周一 09:00 触发。

## 4.6.4 通知联系点

**通知联系点**是一个交付渠道——电子邮件地址列表、Webhook、消息平台集成或其他受支持的渠道——在系统通知设置中配置。

配置规则时，从下拉列表中按名称选择联系点。如果下拉列表中没有联系点，则必须先在通知配置部分创建，才能激活定时报告规则。

请参阅[系统配置](../14-administration/05-system-configuration.md)，了解创建和管理通知联系点的说明。
