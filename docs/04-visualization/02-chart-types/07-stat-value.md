# 4.2.7 统计值

## 概述

统计值面板用于以大字体显示单个数值，并可附加标签和时间戳。它适用于需要在大屏或汇总视图中清晰展示关键指标的仪表板和状态仪表板。

![统计值示例](../images/stat-demo.png)

显示的值为所选时间范围内的最新数据点。字体大小、颜色、背景以及布局均可配置，以匹配所在仪表板的视觉设计。

## 适用场景

在以下情况下使用统计值面板：

- 需要在仪表板上展示核心指标——今日总产量、当前温度、活跃报警数
- 希望在操作员大屏上显示大字体、易于远距离阅读的数值
- 正在构建将多个关键指标并排展示的 KPI 汇总面板

如需对照刻度或范围查看数值，请使用仪表盘图或条形仪表。如需查看历史趋势，请使用趋势图。

## 配置

### 编辑模式工具栏

除[通用编辑模式控件](../01-panels.md#424-面板编辑模式)外，统计值还增加了以下控件：

| 控件 | 说明 |
|---|---|
| **保存为图片** | 将当前预览下载为 PNG 图片 |
| **全屏** | 将编辑器预览扩展为填满浏览器窗口 |
| **解读面板** | 对当前预览数据运行 AI 分析 |

### 图形设置

| 设置 | 说明 |
|---|---|
| **布局** | **水平**（标签与数值并排）或**垂直**（标签在数值上方） |
| **显示时间** | **开启：** 在数值下方显示该数据点的时间戳。**关闭：** 仅显示数值。 |
| **名称字体大小** | 指标名称标签的字体大小（默认 16） |
| **名称字体颜色** | 标签文字颜色 |
| **数值字体大小** | 数值的字体大小（默认 48） |
| **数值字体颜色** | 数值文字颜色 |
| **背景颜色** | 面板背景颜色 |
| **宽度** | 面板的固定像素宽度（留空则自动调整大小） |

## 使用示例

**仪表板核心指标。** 工厂经理的仪表板横向排列三个统计值面板：今日总能耗、当前每小时产量、活跃报警数。每个面板均使用大字体（64），深色背景白色数值，可在控制室远距离清晰读取。

**班次汇总 KPI。** 在班次交接时，统计值面板显示过去 8 小时的总产量。启用"显示时间"功能，操作员可以确认数值最后更新的时间。

**温度监测点。** 关键炉温以统计值形式显示，当属性的上限被超越时，数值字体颜色设置为红色，在仪表板内提供即时视觉警报。
