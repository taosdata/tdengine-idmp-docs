# 4.2.2 统计值

## 4.2.2.1 概述

统计值面板用于以大字体显示单个数值，并可附加指标名称、时间戳和迷你趋势图。适用于需要在大屏或汇总视图中清晰展示关键指标的仪表板场景。

![统计值面板编辑模式概览](../../images/04/stat-demo.png)

面板显示所选时间范围内的最新数据点。每个指标占据一个独立单元格，包含名称标签、数值及时间戳。背景可选择显示面积迷你图，直观呈现数据在时间窗口内的变化趋势。

## 4.2.2.2 适用场景

在以下情况下使用统计值面板：

- 需要在仪表板上醒目展示核心指标——当前电压、电流、功率等实时数据
- 希望在操作员大屏上显示大字体、易于远距离阅读的数值
- 正在构建将多个关键指标并排展示的 KPI 汇总面板
- 需要通过背景色快速反映数值所处状态区间

如需对照刻度或范围查看数值，请使用仪表盘或条形仪表盘。如需查看详细历史趋势，请使用趋势图。

## 4.2.2.3 配置

### 图形配置

图形配置决定统计值单元格的布局与外观：

![统计值面板编辑模式概览](../../images/04/stat-demo.png)

| 设置 | 说明 |
|---|---|
| **布局** | 单元内标签与数值的排列方式：Auto（自动）、Horizontal（名称和数值并排）、Vertical（名称在上数值在下）；默认 Auto |
| **显示时间** | 是否在数值下方显示数据时间戳：On 或 Off；默认 On |
| **时间格式** | 时间戳的显示格式，如 `YYYY-MM-DD HH:mm:ss`（显示时间为 On 时可用） |
| **文本模式** | 单元格内展示的文字内容：Value（仅数值）、Name（仅名称）、Value and Name（数值和名称） |
| **颜色模式** | 阈值颜色的应用位置：Value（为数值着色）、Text（为所有文字着色）、Background Gradient（渐变背景）、Background Solid（纯色背景）；选择背景模式时可设置背景颜色 |
| **图形模式** | 是否在单元格背景显示迷你趋势图：None（无）或 Area（面积图）；默认 Area |
| **文字对齐** | 文字的水平对齐方式：Auto、Left、Center、Right；默认 Auto |
| **名称颜色** | 指标名称标签的文字颜色（文本模式包含名称时可用） |

#### 垂直布局

将布局设为 Vertical 时，每个指标纵向堆叠，名称在上、数值在下，适用于仪表板纵向排列多个指标的场景：

![垂直布局效果](../../images/04/stat-vertical.png)

#### 背景颜色模式

将颜色模式设为 Background Gradient 或 Background Solid 后，每个单元格的背景色由阈值决定，实现按值着色的视觉效果：

![背景渐变颜色模式](../../images/04/stat-background.png)

### 标准配置

| 设置 | 说明 |
|---|---|
| **小数位数** | 数值显示的小数位数（留空则自动判断） |
| **配色方案** | 系列颜色分配策略：单色、单色深浅映射（按系列）、阈值取色（按值）、经典调色板、经典调色板（按系列名）、自定义调色板 |
| **无数据** | 无数据时显示的文本（默认 `-`） |

### 字体大小

通过设置名称和数值的字体大小，可独立控制标签与数字的视觉比重：

![字体大小配置](../../images/04/stat-textSize.png)

| 设置 | 说明 |
|---|---|
| **名称** | 指标名称标签的字体大小（留空则自动计算） |
| **数值** | 数值的字体大小（留空则自动计算） |

### 数据链接

数据链接为统计值单元格附加可点击的跳转 URL：

| 设置 | 说明 |
|---|---|
| **标题** | 链接的显示名称 |
| **URL** | 跳转目标地址，支持变量插值 |
| **在新标签页打开** | 是否在新浏览器标签页中打开链接 |
| **一键跳转** | 启用后点击单元格直接跳转（同时只能有一条链接启用此功能） |

### 值映射

值映射将数据值替换为自定义显示文本并赋予颜色：

| 映射类型 | 说明 |
|---|---|
| **值** | 精确匹配特定数值或文本 |
| **范围** | 匹配指定数值范围 |
| **正则表达式** | 使用正则表达式匹配并替换 |
| **特殊值** | 匹配 null、NaN、布尔值、空字符串等 |
| **其他值** | 匹配所有未被前面规则覆盖的值 |

### 颜色阈值

颜色阈值定义数值区间与颜色的对应关系，配合**颜色模式**将颜色应用于数值文本或单元格背景：

![颜色阈值配置](../../images/04/stat-thresholds.png)

| 设置 | 说明 |
|---|---|
| **阈值模式** | 阈值判断方式：Absolute（绝对值）或 Percentage（最小值–最大值范围的百分比） |
| **添加阈值** | 新增一条阈值规则，每条包含数值边界和对应颜色 |

如上图所示，配置阈值 200（红色）、90（橙色）、5（黄色）、Base（蓝色）后，每个指标的数值和迷你图颜色将根据当前值自动映射到对应的阈值颜色。颜色阈值生效需在标准配置中将**配色方案**设置为**阈值取色（按值）**。

### 个性化配置

个性化配置允许对单个指标覆盖全局设置。选定目标指标名称（Fields with name）后，可覆盖以下属性：系列样式、填充透明度、值映射等。

### 降采样

当查询结果中的数据点过多时，可启用降采样减少计算量：

| 设置 | 说明 |
|---|---|
| **启用降采样** | 开关，默认关闭 |
| **最大数据点数** | 降采样后保留的最大数据点数量 |
| **聚合函数** | 降采样时使用的聚合方式（如 AVG、MAX、MIN 等） |

### 定时报告

定时报告按预设周期自动生成面板快照并推送：

| 设置 | 说明 |
|---|---|
| **频率** | 发送间隔：每周、每天等 |
| **任务开始时间** | 首次执行的日期和时间 |
| **结束日期** | 定时任务终止日期（留空则持续执行） |
| **通知联系人** | 接收报告的通知联系点 |

## 4.2.2.4 使用示例

**仪表板核心指标。** 工厂经理的仪表板横向排列四个统计值面板：电压、电流、功率、相位。将颜色模式设为 Background Gradient，文字对齐设为 Center，配合不同颜色的背景渐变呈现各指标状态，在控制室远距离即可辨识。

**阈值告警看板。** 运维人员配置多个阈值区间（如 200 红色、90 橙色、5 黄色），当电压超过 200 V 时自动显示红色；电流在正常范围时显示黄色。数值颜色变化无需额外告警规则即可提供即时视觉警报。

**大屏垂直布局。** 在竖屏大屏场景中，将布局设为 Vertical、字体大小设为名称 80、数值 40，使每个指标纵向展示。名称标签突出显示，数值使用阈值取色，形成紧凑且易读的纵向 KPI 列表。
