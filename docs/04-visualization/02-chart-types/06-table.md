# 4.2.6 表格

## 概述

表格面板以结构化网格显示查询结果，每个时间点一行，每个指标一列。这是读取查询返回的精确数值最直接的方式——无聚合、无视觉编码，只有数字。

所有配置的指标均显示为列。时间戳列始终包含在内。面板底部的分页控件让您可以在大型结果集中翻页。

## 适用场景

在以下情况下使用表格面板：

- 需要查看精确的数据值而非视觉趋势
- 正在验证数据质量或检查缺失值
- 希望构建带有每个时间桶聚合值的汇总报告
- 需要导出数据进行进一步分析

对于视觉趋势分析，请使用趋势图。对于单个汇总数值，请使用统计值面板。

## 配置

### 编辑模式工具栏

除[通用编辑模式控件](../01-panels.md#424-面板编辑模式)外，表格还增加了以下控件：

| 控件 | 说明 |
|---|---|
| **保存为图片** | 将当前预览下载为 PNG 图片 |
| **全屏** | 将编辑器预览扩展为填满浏览器窗口 |
| **解读面板** | 对当前预览数据运行 AI 分析 |

### 图形设置

| 设置 | 说明 |
|---|---|
| **时间戳格式** | 时间戳列的显示格式（默认：`YYYY-MM-DD HH:mm:ss`） |

表格面板没有坐标轴、边界值或图例部分。

## 使用示例

**数据质量检查。** 数据工程师将电表元素的所有属性添加到表格面板，设置 1 小时时间范围并启用禁用采样。逐行的原始输出让他们验证读数是否按预期间隔到达，以及是否有值缺失或超出范围。

**班次汇总报告。** 运营经理配置带有 1 天滑动窗口和聚合函数（能耗求和、温度求均值）的表格。生成的表格每天显示一行，包含每个指标的聚合值——一份干净的表格报告，适合导出使用。

**事件回顾。** 维护工程师回顾按小时分组的 7 天报警计数表格。时间戳和计数列精确记录了报警活动最高的时段，对趋势可视化形成补充。
