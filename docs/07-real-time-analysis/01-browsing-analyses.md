# 7.1 浏览和管理分析

分析在元素浏览器中任意元素的**分析**标签页中管理。在资产树中导航到某个元素并点击**分析**标签页，即可查看该元素上配置的所有分析。

## 7.1.1 分析列表

分析列表显示当前元素上的所有分析，包含以下列：

| 列 | 说明 |
|---|---|
| **名称** | 分析名称——点击可打开分析详情视图 |
| **触发类型** | 触发类型（滑动窗口、周期窗口等） |
| **流名称** | 此分析对应的 TDengine 流名称 |
| **模板** | 创建此分析所使用的分析模板（如有） |
| **类别** | 类别标签 |
| **状态** | 当前执行状态：**运行中**或**已暂停** |
| **更新时间** | 分析最后修改的时间 |

## 7.1.2 工具栏

| 控件 | 说明 |
|---|---|
| **+** | 手工创建新分析——打开分析创建表单 |
| **粘贴** | 将之前复制的分析粘贴到此元素 |
| **刷新** | 重新加载分析列表 |
| **自动刷新间隔** | 设置自动刷新的下拉菜单：关闭、1秒、5秒、10秒、15秒、30秒、1分钟、5分钟 |
| **导出当前列表为 CSV** | 将分析列表导出为 CSV 文件 |
| **选择列** | 显示或隐藏列表中的列 |

列表上方的过滤区域提供**类别**下拉菜单以按类别标签过滤，以及**AI** 按钮以切换 AI 辅助创建面板。

## 7.1.3 行操作

将鼠标悬停在任意分析行上，点击右侧的 **⋮**（更多）菜单，可访问以下操作：

| 操作 | 说明 |
|---|---|
| **查看** | 打开此分析的只读详情视图 |
| **编辑** | 以编辑模式打开分析以修改配置 |
| **复制** | 复制此分析，可粘贴到另一个元素 |
| **转换为模板** | 将此分析保存为基础库中可复用的分析模板 |
| **趋势图分析** | 打开元素的趋势图，用于检查分析输出 |
| **重算历史** | 对历史数据重新运行分析以回填输出属性 |
| **暂停** | 暂停分析——停止执行但不删除（暂停时显示**恢复**） |
| **删除** | 删除分析，并可选择性删除其产生的输出数据 |

## 7.1.4 分析状态

| 状态 | 含义 |
|---|---|
| **运行中** | 分析流处于活动状态，随着新数据到达持续执行 |
| **已暂停** | 分析已手动暂停——恢复前不会运行新计算 |

删除分析时，确认对话框会询问是否同时删除该分析之前写入的旧输出数据。
