# 7.4 计算

计算区段（分析表单第 3 段）定义分析计算什么以及结果存储在哪里。它包含四个部分：**计算对象**、**窗口聚合**、**输出时间戳**和**输出属性**。

## 7.4.1 计算对象

**计算对象**决定分析的数据范围——即计算针对当前元素自身的属性执行，还是对其下属子元素的数据进行跨元素聚合。

<table>
<colgroup><col style="width:8em"/><col/></colgroup>
<thead><tr><th>选项</th><th>说明</th></tr></thead>
<tbody>
<tr><td><strong>元素本身</strong></td><td>计算针对元素自身的属性执行。适用于针对单台设备或单个测点进行计算的场景。</td></tr>
<tr><td><strong>子元素聚合</strong></td><td>对元素下共享相同模板的子元素数据进行聚合计算。适用于"计算该风电场下所有风机的平均发电功率"等跨子元素的汇总指标。</td></tr>
</tbody>
</table>

选择**子元素聚合**后，会显示两个附加字段：

<table>
<colgroup><col style="width:8em"/><col/></colgroup>
<thead><tr><th>字段</th><th>说明</th></tr></thead>
<tbody>
<tr><td><strong>子元素模板</strong></td><td>子元素必须匹配的模板。仅具有该模板的子元素参与聚合。若所有子元素共享相同模板，系统将自动预填。</td></tr>
<tr><td><strong>子表过滤</strong></td><td>可选的过滤表达式，用于缩小参与聚合的子元素范围。例如，仅筛选处于特定运行状态的子元素。</td></tr>
</tbody>
</table>

:::note
**子元素聚合**仅在元素包含子元素时可用。对于叶节点元素，此选项处于禁用状态，仅可选择**元素本身**。
:::

## 7.4.2 窗口聚合

**窗口聚合**复选框（默认启用）控制计算是否在时间窗口上进行聚合。

启用时，**间隔**字段指定聚合窗口的长度（数字 + 时间单位）。例如，间隔为 1 小时表示每次触发计算过去 1 小时数据的聚合值。

禁用时，计算在单个数据点上运行，不进行窗口聚合——适用于行级计算或转换。

## 7.4.3 输出时间戳

**输出时间戳**下拉菜单指定每个结果行写入输出属性时使用哪个时间戳：

<table>
<colgroup><col style="width:7em"/><col/></colgroup>
<thead><tr><th>选项</th><th>说明</th></tr></thead>
<tbody>
<tr><td><strong>窗口开始</strong></td><td>窗口开始时间</td></tr>
<tr><td><strong>窗口结束</strong></td><td>窗口结束时间（默认）</td></tr>
</tbody>
</table>

**偏移量**字段为所选窗口边界添加时间偏移量（数字 + 单位，默认 0 秒）。可用于调整输出时间戳以对齐显示。

## 7.4.4 输出属性

**输出属性**表格定义计算表达式与元素属性（以及在启用事件生成时与事件属性）之间的映射关系。

表格中每行包含以下列：

<table>
<colgroup><col style="width:7em"/><col/></colgroup>
<thead><tr><th>列</th><th>说明</th></tr></thead>
<tbody>
<tr><td><strong>表达式</strong></td><td>计算表达式。点击单元格打开表达式编辑器（参见<a href="../03-data-modeling/02-attributes.md#329-表达式编辑器">第 3.2.9 节</a>）。</td></tr>
<tr><td><strong>元素属性</strong></td><td>计算结果作为新时序值存储的元素属性</td></tr>
<tr><td><strong>事件属性</strong></td><td><em>（仅在第 4 段启用事件生成时可见）</em> 在事件触发时刻捕获计算值的事件属性</td></tr>
</tbody>
</table>

通过表格底部的 **+** 按钮可添加更多输出行。每行对应一个独立的表达式，支持在单个分析中计算多个指标并将结果写入不同属性。
