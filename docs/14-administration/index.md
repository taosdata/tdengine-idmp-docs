# 14. 系统管理

本章介绍 TDengine IDMP 的系统管理相关内容，包括部署架构、容量规划、用户管理、系统配置以及备份与恢复。

管理控制台通过点击右上角头像并选择**管理控制台**来访问。

| 章节 | 说明 |
|---|---|
| **部署架构** | 单实例与高可用部署拓扑概述 |
| **规划** | 硬件要求、操作系统支持、端口分配及目录结构 |
| **用户管理** | 用户、角色、权限，以及通过 OAuth 2.0 实现单点登录（SSO） |
| **系统配置** | 基本设置、通知联系人、通知模板和邮件服务器 |
| **备份与恢复** | 配置自动备份及从备份中恢复数据 |

:::note
**监控与日志**功能计划在未来版本中提供。
:::
