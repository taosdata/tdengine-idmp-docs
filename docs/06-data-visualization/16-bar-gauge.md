# 条形仪表盘

条形仪表盘以填充条形的方式直观展示指标的当前值在其量程范围内所处的位置，适合用于**实时监控单个或多个指标的当前状态、快速判断指标是否接近或超过阈值**。与仪表盘（圆形仪表）相比，条形仪表盘在同一空间内可展示更多指标。

![条形仪表盘示例](./images/bar-gauge-demo.png)

## 配置项

### 图形配置

#### 标题

可设置面板的主标题和副标题，显示在面板顶部，用于说明图表的主题内容。

#### 布局

条形仪表盘支持**水平**和**垂直**两种布局方式：

- **水平**：条形沿水平方向延伸，标签显示在左侧，适合展示较多指标。
- **垂直**：条形沿垂直方向延伸，标签显示在下方，适合强调单个指标的量级感。

![条形仪表盘布局](./images/bar-gauge-layout.png)

#### 数值显示

控制条形上方数值的显示方式，支持以下三种模式：

- **数值颜色**：数值颜色跟随当前阈值区间的颜色变化，直观反映当前状态。
- **文本颜色**：数值始终以固定文本颜色显示，不随阈值变化。
- **隐藏**：不显示数值，仅显示条形的填充状态。

#### 名称显示

控制指标名称的显示方式：

- **自动**：根据空间大小自动决定是否显示指标名称。
- **隐藏**：始终隐藏指标名称。

#### 条形大小

- **自动**：系统根据面板尺寸和指标数量自动计算每个条形的大小。
- **手动**：手动指定每个条形的宽度（水平布局）或高度（垂直布局），单位为像素。

![条形仪表盘大小配置](./images/bar-gauge-size.png)

#### 数值范围

- **最小值**：设置条形量程的下限，不设置则自动使用数据最小值。
- **最大值**：设置条形量程的上限，不设置则自动使用数据最大值。
- **小数位数**：控制显示数值的小数位数。

### 阈值

阈值用于将量程划分为不同的颜色区间，帮助用户快速判断当前值处于哪个状态区间（如正常、警告、危险）。点击 **+ 添加阈值** 按钮可以添加多个阈值。

每个阈值包含：

1. **数值**：阈值的分界点数值。
2. **颜色**：超过该阈值后，条形的填充颜色。

:::tip
建议配置至少两个阈值，例如：将 80% 以下设为绿色（正常），80%~95% 设为黄色（警告），95% 以上设为红色（危险），使条形颜色直观反映设备运行状态。
:::

![条形仪表盘阈值配置](./images/bar-gauge-threshold.png)

### 通知规则

可为面板配置通知规则，当指标值触发预设条件时，系统将自动发送告警通知。详见通知规则相关文档。

## 配置项增强 roadmap

为了持续提升您的使用体验，我们将在后续的产品更新中，不断增加实用配置，让您能够享受到更丰富、更贴心的分析功能。

| 项目           | 说明                                                                          |
|----------------|-------------------------------------------------------------------------------|
| 渐变填充       | 支持对条形启用渐变填充效果，使视觉过渡更加平滑自然                               |
| 背景填充       | 支持将整个量程区域以半透明背景填充，增强条形的空间感                              |
| 数值单位       | 支持在数值后显示计量单位，方便直观读取数据含义                                    |
| 指标配色方案   | 目前每个指标的颜色由阈值决定；后续可提供默认配色方案，适用于未设置阈值的场景        |
