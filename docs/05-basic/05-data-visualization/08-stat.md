# 统计值

统计值面板用来显示单一的数值。

![统计值示例](./images/stat-demo.png)

## 配置项

### 图形配置

#### 布局

当有多个统计值时，通过布局可以决定优先按照`水平`或者`垂直`方向排列，默认为`水平`。

#### 名称字体颜色和大小

可设置数值上方的名称文字颜色及其大小。

#### 数值字体颜色和大小

可设置数值文字颜色及其大小。

#### 背景颜色

可设置背景颜色。

#### 宽度

可设置数值板块的显示宽度，默认由数值内容决定宽度。
