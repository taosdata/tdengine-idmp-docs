# 事件列表

事件列表使用表格用来展示企业的事件信息。事件列表作为面板的一种类型，可以单独保存为某个元素下的事件列表面板，也可以添加到仪表板中。

## 保存事件列表面板

点击“事件”主菜单，点击左侧的“事件过滤器”，进入事件查询页面，输入过滤条件，查询关注的事件。如图，点击“另存为事件列表”按钮，将当前的查询结果保存为事件列表面板。

![事件列表入口](images/table-event-01.png)

配置好查询条件，选择要显示的列后，可以点击“另存为面板”按钮，将当前的查询结果保存为事件列表面板。事件列表保存成功后，立即自动打开面板预览页面。也可以进入到保存元素的“面板”标签页，查看新保存的事件列表面板。
