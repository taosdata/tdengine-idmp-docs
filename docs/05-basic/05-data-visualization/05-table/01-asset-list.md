# 资产列表

资产列表使用表格用来展示企业的资产信息及其最新状态和采集值。

## 配置项

### 配置入口

1.将“元素查询结果”转换为资产列表

![资产列表入口](../images/table-asset-sr.png)

2.子元素列表转换为资产列表

![资产列表入口](../images/table-asset-childel.png)

### 资产列表配置

如图，可以对资产列表进行如下配置：

* **查询条件**：其中的资产类型（模版）必填，只有指定了资产类型，才能选择对应的属性字段。
* **显示字段**：可配置要显示的字段及其顺序。字段包括 IDMP 资产自身管理属性，也包括引用属性（TDengine Tags 和 TDengine Metrics）。

![资产列表配置](../images/table-asset-config.png)

根据转换来源的不同，保存逻辑如下：

* **通过 `子元素列表` 转换**：面板将直接保存到当前元素下，可在当前元素面板下查询到。
* **通过 `元素查询结果` 转换**：需要手动指定保存位置，保存后成为指定元素下的面板。

![资产列表保存](../images/table-asset-save.png)

### 资产列表面板

资产列表保存成功，在面板中可以看到资产列表的展示效果，如图所示：

![资产列表面板](../images/table-asset-panel.png)

资产列表面板支持修改、删除等，也支持添加到仪表板中。
