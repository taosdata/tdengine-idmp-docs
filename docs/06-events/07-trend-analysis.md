# 6.7 分析面板

事件发生时，了解事件前后的数据行为对于调查至关重要。TDengine IDMP 提供从任意事件直接跳转到预设事件时间范围分析面板的快捷路径，便于快速分析事件期间的数据变化。

## 6.7.1 从事件打开分析面板

系统提供两种从事件快速打开分析面板的方式，分别适用于列表浏览和详情查看场景。

**从事件列表：**

将鼠标悬停在事件行上显示 **⋮** 菜单，然后点击**添加到分析面板**。IDMP 将在新窗口打开关联事件的分析面板，并自动缩放至事件的开始时间和结束时间。

**从事件详情页：**

在通用标签页，点击工具栏中的**添加到分析面板**图标，打开相同的分析面板视图。

## 6.7.2 使用分析面板

分析面板是 IDMP 高级分析功能的主要入口。

分析面板支持多事件叠加对比、跨元素属性添加及时间窗口调整等分析操作。从事件打开的分析面板显示元素在事件时间窗口内的时序属性，并在图表上标注事件的开始和结束时间。

支持将多个事件添加到同一分析面板中，从而并排叠加对比不同事件——例如对比两次独立故障期间的传感器行为。

在分析面板视图中，支持添加来自该元素或资产树中其他元素的其他属性，并通过缩放或平移调整时间窗口。

:::
