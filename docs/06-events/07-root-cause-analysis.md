# 6.7 根因分析

当事件发生时,快速定位问题根本原因对于减少停机时间和防止问题复发至关重要。TDengine IDMP 提供根因分析功能,帮助操作人员系统化地追溯事件触发链条,识别导致问题的初始原因。

## 6.7.1 打开根因分析

系统提供两种从事件快速打开根因分析的方式。

**从事件列表：**

将鼠标悬停在事件行上显示 **⋮** 菜单,然后点击**根因分析**。系统打开该事件的根因分析视图。

**从事件详情页：**

在通用标签页,点击工具栏中的**根因分析**按钮,打开相同的视图。

## 6.7.2 根因分析报告

AI 将基于当前事件的关键数据，生成一份结构化的、易于理解的、具有业务洞察力的专属分析报告，帮助用户快速排查事件产生的根本原因，降低运维成本。
