# 6.6 趋势图分析

事件发生时，了解事件前后的数据行为对于调查至关重要。TDengine IDMP 提供从任意事件直接跳转到预设了事件时间范围的趋势图的快捷路径。

## 6.6.1 从事件打开趋势图

可通过两种方式为任意事件打开趋势图：

**从事件列表：**

将鼠标悬停在事件行上显示 **⋮** 菜单，然后点击**趋势图分析**。IDMP 打开关联元素的趋势图，并自动缩放至事件的开始时间和结束时间。

**从事件详情页：**

在通用标签页，点击工具栏中的**趋势图分析**图标，打开相同的视图。

## 6.6.2 使用事件趋势图

从事件打开的趋势图显示元素在事件时间窗口内的时序属性，并在图表上标注事件的开始和结束时间。

可将多个事件添加到同一趋势图分析中，从而并排叠加对比不同事件——例如对比两次独立故障期间的传感器行为。

在图表视图中，可添加来自该元素或资产树中其他元素的其他属性，并通过缩放或平移调整时间窗口。

:::note
趋势图分析是一项功能强大的特性，具有许多能力。**高级分析**章节中有专门的完整介绍。
:::
