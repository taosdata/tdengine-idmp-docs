# 6.4 事件详情

在全局事件视图或元素的事件标签页中点击事件名称，即可打开事件详情页。详情页顶部为**通用信息**——展示事件的所有标准字段并提供操作工具栏；底部为五个标签页：**属性**、**关联属性**、**注释**、**通知记录**和**子事件**。

## 6.4.1 通用信息

通用信息区展示事件的所有标准字段，并提供确认、加入分组、根因分析和趋势图分析等操作入口。

### 6.4.1.1 工具栏

通用信息区的工具栏提供以下操作控件。

| 控件 | 说明 |
|---|---|
| **返回列表** | 返回事件列表 |
| **确认** | 确认事件 |
| **加入分组** | 将此事件加入一个或多个自定义分组，可从左侧边栏快速访问 |
| **根因分析** | 打开此事件的根因分析视图 |
| **趋势图分析** | 打开关联元素在事件时间范围内的趋势图 |
| **重新发送** | 手动为此事件向已配置的联系途径重新发送通知 |

### 6.4.1.2 字段

通用信息区展示以下标准事件字段，涵盖事件的时间、关联对象、触发上下文和状态信息。

| 字段 | 说明 |
|---|---|
| **名称** | 事件名称，由事件模板的命名规则生成；子事件名称在父事件名基础上追加触发器名后缀 |
| **模板** | 创建此事件所使用的事件模板 |
| **严重程度** | 严重等级 |
| **原因代码** | 原因代码（如已设置） |
| **分类** | 类别标签 |
| **描述** | 自由文本描述 |
| **操作原因** | 触发此事件的具体原因——对普通事件即触发该分析的条件，对子事件即命中的触发器及其表达式 |
| **当前值** | 事件触发瞬间命中条件的属性取值 |
| **开始时间** | 事件开始的时间 |
| **结束时间** | 事件结束的时间（仍处于活动中则为空） |
| **关联元素** | 触发此事件的分析所涉及的元素，点击可导航；一次分析可能横跨多个元素，因此此字段为元素列表 |
| **关联分析** | 生成此事件的分析规则——点击可导航至该分析 |
| **状态** | 确认状态（未确认 / 已确认） |
| **确认人** | 确认人与确认事件（如有） |

## 6.4.2 底部标签页

通用信息下方提供五个标签页，用于查看事件的扩展信息——事件属性、关联属性、操作注释、通知投递记录以及子事件列表。

### 6.4.2.1 属性

**属性**标签页展示分析在创建事件时一并捕获到事件记录上的命名值——也就是流计算窗口配置的输出项中**写入事件属性**的那部分。这些属性由[事件模板](./01-event-templates.md)定义、由触发事件的分析填充，常用于记录事件发生时的关键数据，例如超限期间的峰值温度或故障时的批次 ID。

| 列 | 说明 |
|---|---|
| **名称** | 属性名称，如事件模板中所定义 |
| **值** | 事件创建时记录的值 |
| **值类型** | 值的数据类型 |

属性值为只读——在事件创建时设定，之后不可修改。工具栏支持按类别过滤、刷新、选择显示列。

### 6.4.2.2 关联属性

**关联属性**标签页展示分析窗口的相关元素属性，以及窗口计算输出项中**写入元素属性**的那部分输出信息——即分析在产生事件的同时，把窗口汇聚计算结果回写到关联元素的属性列表。

事件属性和计算输出属性来自同一份窗口计算输出配置，区别在于写入目标：

- **事件属性**随事件存储，仅在事件记录上可见；
- **关联属性**进入元素的时序数据，可被仪表盘、分析面板等下游消费。

在事件详情页的关联属性标签页上，每条记录展示属性名称、具体取值以及值类型，便于在同一处查看"事件留痕"和"时序留痕"的双份产物。

### 6.4.2.3 注释

注释标签页供操作人员添加与事件相关的文本注释——例如调查发现或已采取的纠正措施。详情参见[注释](../11-collaboration/02-annotations.md)。

### 6.4.2.4 通知记录

通知记录展示此事件所有通知发送的完整日志。每条记录显示联系途径名称、发送时间戳以及投递状态。

### 6.4.2.5 子事件

**仅父事件可见**——列出此父事件下的全部子事件，列与[浏览事件](./03-browsing-events.md)中的事件列表一致；点击任一行可跳转到对应子事件的详情页。父子事件的产生规则、命名规则、时间范围参见 [6.2 子事件](./02-child-events.md)。
