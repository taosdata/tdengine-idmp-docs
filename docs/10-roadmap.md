# 产品路线图

TDengine IDMP 还在积极的开发中，2026 年年度路线图如下表所示。

| 季度 | 功能 |
|------|------|
| 2026Q1 | <ol><li>支持根因分析</li><li>支持手动输入或 CSV 上传采集的数据</li><li>支持事件统计分析面板</li><li>更多类型的面板 (Status History, Candle stick, Heatmap, Histogram, FlameGraph)</li></ol> |
| 2026Q2 | <ol><li>支持面板和仪表板上钻和下钻</li><li>支持仪表板提供各种组件和关联面板</li><li>支持配置信息的版本控制</li><li>支持内部表</li><li>支持子事件</li></ol> |
| 2026Q3 | <ol><li>支持面板和仪表板主题选择和可定制</li><li>支持数据质量监测</li><li>支持第三方图形插件</li><li>支持面板图例，系列，坐标轴等更多可配置项</li></ol> |
| 2026Q4 | <ol><li>提供系统可观察指标</li><li>支持用户组</li><li>支持关系库</li><li>支持第三方时序库</li></ol> |
