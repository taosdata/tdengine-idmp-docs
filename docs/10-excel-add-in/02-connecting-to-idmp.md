# 10.2 将 Excel 连接到 IDMP

安装 Excel 插件后，您需要在 Excel 中打开它并登录到 IDMP 实例，然后才能检索数据。

## 打开插件

TDengine IDMP 插件是一个 Office Web 加载项。安装后，它不会自动显示为功能区选项卡——首次使用时需要从**我的加载项**中激活。

1. 打开 Microsoft Excel。
2. 点击功能区中的**插入**选项卡。
3. 在**加载项**组中，点击**我的加载项**。

   :::tip
   如果看不到**我的加载项**按钮，请在"插入"选项卡中查找小型**加载项**图标，或点击"加载项"按钮上的下拉箭头。在较旧的 Excel 版本（2016/2019）中，该按钮可能标注为 **Office 加载项**。
   :::

4. 在 **Office 加载项**对话框中，选择**我的加载项**选项卡。
5. 在列表中找到 **TDengine IDMP**，双击它，或选中后点击**添加**。

TDengine IDMP 任务窗格将在 Excel 窗口右侧打开。

:::note
首次激活后，Excel 可能会记住该加载项，并在后续会话中直接在**主页**选项卡或专用功能区选项卡中显示 **TDengine IDMP** 按钮。如果重启 Excel 后加载项消失，请重复上述步骤，从**我的加载项**重新打开它。
:::

## 加载项未显示在"我的加载项"中

如果安装后 **TDengine IDMP** 未出现在**我的加载项**列表中：

- **关闭并重新打开 Excel** — 加载项注册仅在完全重启后生效。
- **检查是否被安全策略阻止（Windows）** — 从互联网下载的文件可能被 Windows 阻止。如果安装脚本下载了任何文件，请右键单击每个文件，选择**属性**，然后勾选"常规"选项卡底部的**解除锁定**。
- **确认安装已完成** — 重新运行[安装 Excel 插件](./01-installation.md)中的安装命令，并检查是否有错误输出。

## 登录

任务窗格打开后：

1. 以 `https://<host>:<port>` 格式输入 IDMP 服务器地址（例如 `https://idmp.tdengine.net:6034`）。
2. 输入您的 IDMP 用户名和密码。
3. 点击**登录**。

连接成功后，任务窗格将显示您的 IDMP 资产层级，并允许您浏览元素及其属性。

## 连接要求

- IDMP 服务器必须可从运行 Excel 的机器访问。
- IDMP 服务器上必须配置 HTTPS（请参阅[安装 Excel 插件](./01-installation.md)中的 HTTPS 设置）。
- 您的 IDMP 账户必须对要检索的元素和属性至少具有读取权限。
