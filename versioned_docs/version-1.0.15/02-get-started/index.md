# 2. 快速上手

欢迎使用 TDengine IDMP。无论您是希望快速体验产品，还是准备在团队中正式落地，这里都有适合您的起步方式。

TDengine IDMP 支持三种部署与访问方式：全托管云服务、基于 Docker 的本地部署，以及通过安装包直接安装。三种方式最终都将为您提供一个已就绪的 IDMP 实例，内置示例数据，让您立即开始探索工业数据的智能价值。

如果您所在的组织已经部署了 TDengine IDMP（无论是云端还是本地），则无需自行安装配置。请联系管理员，请其向您发送邀请邮件。邮件中包含账号激活链接，点击链接并设置密码后，即可直接进入 IDMP Web 控制台，请阅读[第 2.4 节](./04-experiencing-idmp.md)，立即探索 IDMP 功能。
