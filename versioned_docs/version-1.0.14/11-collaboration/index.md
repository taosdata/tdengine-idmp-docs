# 11. 协作与共享

工业数据的价值，只有在组织内部被充分共享和协作时才能得到最大程度的发挥。单个工程师掌握的洞察，若无法便捷地传递给管理层、运维团队或跨地区的同事，其价值便会大打折扣。TDengine IDMP 将协作能力内置于平台核心，让团队能够共享可视化内容、留存现场知识、导出数据，以及在不同系统或部署环境之间迁移数据模型。

本章涵盖 IDMP 中所有与协作和共享相关的功能：共享仪表板与面板、在元素和事件上添加注释、将数据导出为 CSV，以及在不同 IDMP 实例之间导入和导出数据模型。
